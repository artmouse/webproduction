Лицензия OneBox

Лицензия OneBox выдается на определенное количество пользователей.
Для защиты от нелицензионного использования OneBox существует система проверки, которая работает по следующему принципу:
  • Каждой копии OneBox выдается уникальный ключ.
  • На сайте license.onebox-system.com данный ключ помещается в файл keys.xml в таком виде:

    <pair>
    <key>УНИКАЛЬНЫЙ КЛЮЧ ONEBOX</key>
    <count>КОЛИЧЕСТВО ПОЛЬЗОВАТЕЛЕЙ</count>
    </pair>
    Например:

    <pair>
    <key>11111-22222-33333-44444</key>
    <count>2</count>
    </pair>
  • В проданной копии OneBox данный ключ записывается в конфиг-параметры движка.

    Engine::Get()->setConfigField('box-key', '11111-22222-33333-44444');
  • Каждый час в проданной копии OneBox выполняется запрос к license.onebox-system.com, по которому в ответ на ключ возвращается разрешенное количество пользователей.
  • Каждую минуту выполняется проверка, сколько пользователей используют копию OneBox, и если это число превышает заданное, "лишние" пользователи "кикаются" и теряют свой уровень доступа в систему.
  • Если ключ не задан, разрешенное количество пользователей — 1.
  • Если ключ задан, но не прописан на license.onebox-system.com, разрешенное количество пользователей — 0.
