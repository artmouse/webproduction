Обычные формы

Обычные формы - это вариант разметки форм в системе OneBox.
Он позволяет выводить форму в табличном виде с минимальным набором опций. Слева название элемента формы, справа - результат.
Для отображения дополнительно текста необходимо использовать
.light

Для выравнивания названия элемента по верхнему краю необходимо использовать
.vtop


.ob-block-form
<div class="ob-block-form">
    <table>
        <tr>
            <td class="vtop">Example caption</td>
            <td>
                <input type="text" value="">
                <div class="light">description text</div>
            </td>
        </tr>

        <tr>
            <td>Example caption</td>
            <td>
                <select class="chzn-select">
                    <option value="">Example value 1</option>
                    <option value="">Example value 2</option>
                    <option value="">Example value 3</option>
                    <option value="">Example value 4</option>
                    <option value="">Example value 5</option>
                </select>
            </td>
        </tr>

        <tr>
            <td class="vtop">Example caption</td>
            <td><textarea></textarea></td>
        </tr>
    </table>
</div>
