Как можно задать размер картинки в конфиге



Зачем это нужно?

Для того, чтобы не менять размер картинки в карточке товара для каждого custom-проекта,
был создан конфиг для задания этих размеров


Как это работает?

Если Вы хотите задать размер картинок в карточке товара,
то для этого вам необходимо в файле engine.mode.php вписать строку 


/* Задать размер картинки в карточке товара
 *
 * 330 - ширина картинки
 * 225 - высота картинки
 *
*/
Engine::Get()->setConfigField('product-image-size', array(330, 225));

Что мы получим в результате?

После этого, в карточке товара, картинки будут отображаться с заданными размерами.
В инном случае картинки будут отображаться в стандартном размере 330 * 225.
