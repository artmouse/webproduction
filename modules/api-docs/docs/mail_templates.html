Как создавать шаблоны для писем в OneBox.

Что это такое шаблоны для писем?

Шаблоны для писем - это HTML-код заготовки письма для рассылки.


Как осуществлять отправку массовой рассылки?

Осуществлять отправку массовой рассылки можно следующим образом:
  • Загрузка шаблона на сервер.
  • Прямая вставка HTML-кода.

Загрузка шаблона на сервер.

  1. Необходимо подготовить HTML-шаблон письма, предварительно вставив в него переменную [content] (на месте переменной будет отображаться текст отправляемого письма).
  2. Загрузить шаблон по адресу \media\mail-design\.
  3. Перейти на страницу массовой рассылки.
  4. Заполнить необходимые поля (тема письма, обратный адрес, дата отправки, текст письма).
  5. В выпадающем списке "Шаблон письма" выбрать загруженный шаблон.
  6. Тип отправки переключить на "HTML-письмо" (в противном случае письмо придет в виде текста).
  7. Отправить письмо.

Прямая вставка HTML-кода.

  1. Перейти на страницу массовой рассылки.
  2. Заполнить необходимые поля (тема письма, обратный адрес, дата отправки).
  3. Тип отправки переключить на "HTML-письмо" (в противном случае письмо придет в виде текста).
  4. В тексте письма нажимаем кнопку "Показать источник".
  5. В поле источника вставляем HTML-код письма.
  6. В тексте письма нажимаем кнопку "Скрыть источник".
  7. Отправить письмо.
