Как добавлять кастомные баннера в OneBox

Регистрация баннера в системе

Для того, чтобы кастомные баннера появились на сайте их необходимо зарегистрировать в include.php в корне текущего проекта. 

$a = array();
$a['banner_id'] = array(
    'name' => 'Баннер на главной 490x240',
    'width' => 490,
    'height' => 240,
    'method' => 'crop'
);

Engine::Get()->setConfigField(
    'shop-banner-place', $a
);
Где:
banner_id - идентификатор баннера,
name - название баннера,
width - ширина баннера,
height - высота баннера,
method - метод обработки изображения (prop / crop / false).

При использовании метода prop, ширина и высота баннера будут масштабироваться пропорционально в рамках заданной ширины и высоты.
При использовании метода crop, ширина и высота баннера будут масштабироваться и отсекаться.
При использовании метода false, ширину и высоту баннера указывать не нужно, так как изображение не будет обрабатываться и будет выводиться оригинал.


Визуальный вывод баннера

Название массива будет иметь вид banner_идентификатор.
Содержимое массива name, url, image и comment.

{|foreach from=$banner_banner_id item="e"|}
    <a href="{|$e.url|}">{|$e.name|}</a>
    <img src="{|$e.image|}" alt="">
    {|$e.comment|}
{|/foreach|}
После регистрации кастомного баннера, стандартный набор баннеров не будет доступен в системе.
