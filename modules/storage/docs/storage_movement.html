Переместить продукты

CRM система OneBox также поддерживает возможность перемещать продукты со склада на склад. Важно! Переместить можно только те товары, которые уже есть на складе, в противном случае система выдаст ошибку «Недостаточно товара на складе».


Для того, чтобы приступить к перемещению товаров нужно активировать меню OneBox, которое находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Склады» → «Переместить продукты».




Далее появится страница для выбора, с какого склада будет произведено перемещение товаров:




Выберите склад в выпадающем списке «С какого склада», затем нажмите на кнопку «Выбрать».
После этого на экране появится страница для перемещения, которая выглядит следующим образом:




Перемещение продуктов, как и оприходование, происходит через корзину.
Товары можно добавлять в корзину вручную из списка всех товаров. Для этого следует воспользоваться блоком «Добавить продукт в корзину». Введите название продукта в поле ввода рядом с кнопкой «Выбрать/найти/создать продукт»:




В выпадающем списке будут отображены все возможные варианты продуктов, которые на данный момент существуют в базе. Если выпадающий список не появился, значит такого продукта не существует.
Для выбора продукта нужно кликнуть по его названию. Выбранный продукт будет отображен на экране сразу над полем для ввода. Таким образом, можно выбирать наименования поочереди, добавляя их в корзину и формируя новый список. После формирования корзины нажмите на кнопку «Сохранить изменения».




Для просмотра списка всех продуктов нужно нажать на кнопку «Выбрать/найти/создать продукт...».




Здесь отображается информация о номере, названии, наличии и стоимости продукта.
В отличии от быстрого поиска данная страница позволяет фильтровать поиск по категории. Это полезно в том случае, если известно неточное название продукта или неизвестно вообще.
В остальном поиск происходит аналогично. В поле «Найти продукт» нужно указать точное или приблизительное имя продукта, после чего нажать на кнопку «Найти»:




Для выбора продукта нужно нажать на него левой кнопкой мышки. После этого система автоматически выполнит переход на предыдущую страницу, курсор мышки установится в поле поиска продукта рядом с его номером. Теперь для добавления продукта в список нужно нажать на клавишу «enter».




На странице просмотра всех продуктов также есть возможность создавать новые продукты.




Для этого вместо кнопки «Найти продукт» нужно выбрать кнопку «Создать продукт». После чего на экране появится форма создания нового продукта.




«Название продукта» является единственным обязательным полем для заполнения. Если продукт не имеет названия - система не сможет его создать, а на экране появится модальное окно с надписью name.
Все остальные поля являются уточняющими.
После заполнения всех полей нажмите на кнопку «Создать».


Дальше система автоматически выполнит переход на предыдущую страницу, курсор мышки установится в поле поиска продукта рядом с его номером. Теперь для добавления продукта в корзину нужно нажать на клавишу «enter».


Каждый товар, который ложится в корзину имеет следующие параметры:

  • количество
  • серийный номер (для одиночных товаров)
  • привязано продукта
  • удалить

Вышеуказанные поля можно редактировать.
Ссылка «Привязать к продуктам на складе» позволяет определить, какой именно продукт будет продаваться/перемещаться. Это полезно, если на складе присутствует множество одинаковых продуктов с разной стоимостью. Если не использовать привязку, система автоматически привяжет самый старый товар со склада (по принципу FIFO). Используя привязку, у пользователя появляется возможность "вырвать" товар из стандартного потока.
Также из корзины можно удалять товары для этого служит checkbox «удалить». После внесения любых изменений в корзину нужно обязательно подтверждать их кнопкой «Сохранить изменения». Система сигнализирует о сохранении изменений специальным уведомлением:




История перемещений товаров фиксируется в журнале. При помощи журнала можно не только следить за перемещением товаров, но и так же удалять действия, совершенные на складах. Например, это может пригодиться, если при перемещении была допущена ошибка. Подробней о журнале можно прочесть в разделе «Склады» → «Журнал» или по этой ссылке .


Блок «Переместить продукты» дает возможность выбрать склад, куда будут перемещены продукты, а также указать дату перемещения. Этим блоком нужно пользоваться, когда корзина продуктов заполнена.


После того, как все поля формы были заполнены нужно нажать на кнопку «Переместить». На экране появится модальное окно для подтверждения. Далее можно будет увидеть страницу журнала.
