Оприходование продуктов

Оприходование товаров позволяет управлять процессом поступления товаров на склады.
Для того, чтобы приступить к оприходованию товаров нужно активировать меню OneBox, которое находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Склады» → «Оприходование продуктов».




Страница для оприходования выглядит следующим образом:




Процесс оприходования можно разделить на следующие этапы:

  • Заполнение корзины - формирование списка товаров, которые будут добавлены на склад.
  • При необходимости - редактирование этого списка.
  • Выбор поставщика товара и склад, куда эти товары будут помещены.
  • Само оприходование.

Товары можно добавлять вручную из списка всех товаров. Для этого следует воспользоваться блоком «Добавить продукт в корзину». Введите название продукта в поле ввода рядом с кнопкой «Выбрать/найти/создать продукт»:




В выпадающем списке будут отображены все возможные варианты продуктов, которые на данный момент существуют в базе. Если выпадающий список не появился, значит такого продукта не существует.
Для выбора продукта нужно кликнуть по его названию. Выбранный продукт будет отображен на экране сразу над полем для ввода. Таким образом, можно выбирать наименования поочереди, добавляя их в корзину и формируя новый список. После формирования корзины нажмите на кнопку «Сохранить изменения».




Для просмотра списка всех продуктов нужно нажать на кнопку «Выбрать/найти/создать продукт...».




Здесь отображается информация о номере, названии, наличии и стоимости продукта.
В отличии от быстрого поиска данная страница позволяет фильтровать поиск по категории. Это полезно в том случае, если известно неточное название продукта или неизвестно вообще.
В остальном поиск происходит аналогично. В поле «Найти продукт» нужно указать точное или приблизительное имя продукта, после чего нажать на кнопку «Найти»:




Для выбора продукта нужно нажать на него левой кнопкой мышки. После этого система автоматически выполнит переход на предыдущую страницу, курсор мышки установится в поле поиска продукта рядом с его номером. Теперь для добавления продукта в список нужно нажать на клавишу «enter».




На странице просмотра всех продуктов также есть возможность создавать новые продукты.




Для этого вместо кнопки «Найти продукт» нужно выбрать кнопку «Создать продукт». После чего на экране появится форма создания нового продукта.




«Название продукта» является единственным обязательным полем для заполнения. Если продукт не имеет названия - система не сможет его создать, а на экране появится модальное окно с надписью name.
Все остальные поля являются уточняющими.
После заполнения всех полей нажмите на кнопку «Создать».


Дальше система автоматически выполнит переход на предыдущую страницу, курсор мышки установится в поле поиска продукта рядом с его номером. Теперь для добавления продукта в корзину нужно нажать на клавишу «enter».


Каждый товар, который ложится в корзину имеет следующие параметры:

  • количество
  • серийный номер (для одиночных товаров)
  • гарантия
  • цена закупки
  • код партии

Вышеуказанные поля можно редактировать. Также из корзины можно удалять товары для этого служит checkbox «удалить». После внесения любых изменений в корзину нужно обязательно подтверждать их кнопкой «Сохранить изменения». Система сигнализирует о сохранении изменений специальным уведомлением:




История оприходования товаров фиксируется в журнале. При помощи журнала можно не только следить за перемещением товаров, но и так же удалять действия, совершенные на складах. Например, это может пригодиться, если при оприходовании была допущена ошибка. Подробней о журнале можно прочесть в разделе «Склады» → «Журнал» или по этой ссылке .


Блок «Добавить продукты в корзину из XLS-файла» дает возможность загрузить в систему предварительно составленный список продуктов в формате XLS.
Поле «Файл» служит для прикрепления файла к форме.
«Загрузить продукты» - добавляет товары в корзину для дальнейшего оприходования на складе.


Блок «Оприходовать продукты» служит для выполнения оприходования.
Воспользоваться им нужно в самом конце, когда корзина товаров заполнена. Среди полей блока можно выделить следующие:
«Поставщик» - выбор поставщика.
«Оприходовать на склад» - выбор склада, на который будут оприходованы товары.
«Юридическое лицо» - нужно для формирования документов.


После того, как все поля формы были заполнены нужно нажать на кнопку «Провести оприходование». На экране появится модальное окно для подтверждения. Далее можно будет увидеть страницу журнала складов.


Если оприходование происходит на основании заказа, то каждый товар будет привязан к этому заказу. После оприходования товара, этот товар в заказе будет считаться обработанным и система не сможет оприходовать его повторно.


Система позволяет автоматизировать процесс оприходования на складе при помощи бизнес-процессов. Например, заказ, который строится по бизнес-процессу «Закупка у поставщика» может иметь этап «Заказ прибыл». В плане действий этапа в свою очередь добавлено действие «Приходовать заказ на склад». Это позволит оприходовать товары, добавленные в заказ, при переходе на конкретный этап. Именно так достигается автоматизация при работе со складами. Подробней о работе с бизнес-процессами можно прочесть в разделе «Бизнес-процессы» или по этой ссылке .


Если во время оприходования данные товара были изменены, например, цена или количество товара, то эти изменения будут синхронизированы с заказом: изменения продублируются в списке продуктов, добавленных в заказ.
