Добавить правило наценки

Для добавления нового правила нужно кликнуть на вкладку «Добавить правило», которая находится в верхней части экрана.




Форм для добавления правила выглядит так:




«Категория» - поле для указания названия категории, для которой будет применяться новое правило. Названия можно ввести с клавиатуры или же выбрать из выпадающего списка.
В поле можно добавить несколько категорий, так как правило может применяться ко многим категориям.
Чтобы убрать категорию из поля, нажмите на знак «х» рядом с названием категории.




Ниже находится таблица для формирования новых правил. Одна строка в таблице соответствует одному правилу.
Поля «Цена от» и «Цена до» - задают диапазон для которого следует использовать наценку. Желательно всегда четко задавать границы диапазона. Даже если верхней границы не существует, желательно все равно задать максимально большое число, к примеру, 999999.
«Тип наценки» - проценты или сумма.
«Значение наценки» - параметр зависит от типа наценки.
«Валюта» - список всех валют в системе. Настройка функционала происходит в OneBox меню: «Настройки» → «Валюты».
«Бренд» - установка наценок для отдельных брендов.
«Поставщик» - выбор поставщика, для которого формируется наценка. Добавление поставщиков в системе происходит через OneBox меню «Настройки» → «Поставщики».
«Приоритет» - порядок, в котором система будет формировать наценки для товаров. Этот инструмент позволяет накладывать правила слоями. Например, правило с более высоким приоритетом пересчитает цены для всей категории, а другое правило, с более низким приоритетом, будет использовано для пересчета цен конкретного бренда в этой же категории.


После того как все значения были внесены в таблицу, нажмите на кнопку «Сохранить правила», которая находится внизу экрана. Система выполнит переход к странице наценок, где в блоке «Правила наценок для категории» будет отображаться новое правило.
