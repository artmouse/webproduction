Глобальные настройки

Настройки наценок и пересчета цен находятся в глобальных настройках. Для этого нужно активировать меню OneBox, которое находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Настройки».




Далее в открывшемся списке нужно найти пункт «Глобальные настройки» и кликнуть по нему.




Страница глобальных настроек выглядит следующим образом:




Настройки отображаются списком и находятся в блоке «Настройки системы». В блоке также имеется поле для быстрого поиска. Начните вводить в поле символы и список настроек сразу же изменится.


Найдите в списке пункт «Пересчет цен и наличия», нажмите на него левой кнопкой мышки.


Страница для пересчета цен и наличия выглядит так:




Поле «Автоматический пересчет цен» - позволяет задать время выполнения автоматического пересчета цен. Данный пункт является одним из ключевых при автоматизации работы системы.
Checkbox «Округление цен» - отвечает за округление цен.
По умолчанию пересчет цен работает только для товаров, которые есть в наличии. Checkbox «Пересчитывать цены товаров, которых нет в наличии» - позволяет изменить это поведение.
«Приоритет выбора цены» - служит для смены приоритета при выборе цены. Приоритет можно установить в пользу поставщиков или склада.


Для сохранения всех изменений нажмите на кнопку «Сохранить», которая находится в нижней части экрана.
