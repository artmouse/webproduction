Массовая рассылка

В CRM системе OneBox доступно два способа рассылки - email и sms.


Рассылка электронных писем доступна по умолчанию, в то время как рассылка SMS-сообщений - только при интеграции с сервисом TurboSMS.ua.


Кнопки массовой рассылки находятся в пункте «Контакты». Активируйте выдвижную панель, которая находится в левой части экрана, затем нажмите левой кнопкой мышки на пункт «Контакты».




Далее следует пролистать страницу контактов в самый низ.

Из перечня полей формы можно выделить следующие:

  • Поле для темы письма, шаблон, в который будет вставлено письмо и адрес отправителя
  • Текстовый режим и HTML-редактор для написания письма
  • Возможность вставки Ф.И.О. и других контактных данных в текст письма для каждого получателя. Это реализовано с помощью переменных, которые описаны в блоке «Можно использовать вставки»
  • К письму можно прикрепить последние новости из системы, а также файлы

Подробней о шаблонах для писем можно почитать по этой ссылке .


В правом верхнем углу страницы указано количество пользователей, подписанных на рассылку. Письма будут отправлены только им. Данная опция настраивается при помощи checkbox`а «Подписан на рассылку» в профиле каждого контакта. Такой подход позволяет ограничить круг пользователей, подписанных на рассылку.




HTML-редактор включает в себя базовый функционал для форматирования текста.




Для отправки рассылки необходимо нажать на кнопку «Оправить».
Чтобы проверить корректно ли работает отправка писем можно воспользоваться тестовой отправкой. В этом случае письмо будет отправлено только отправителю. Его можно будет просмотреть и проверить правильно ли оно составлено.


Массовая рассылка SMS

Важно! Данный функционал будет работать только в том случае, если к системе подключен сервис TurboSMS.ua. В противном случае сначала нужно произвести интеграцию OneBox и TurboSMS.


За массовую рассылку SMS-сообщений отвечает кнопка «Массовая рассылка SMS», которая находится на странице всех контактов.

Важно! Код телефона в карточке контакта должен быть записан в формате 380XXXXXXXXX, то есть без использования скобок, тире и пробелов. В противном случае сервис TurboSMS.ua не сможет выполнить отправку сообщения на данный номер.
