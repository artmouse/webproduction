Добавление шаблонов для писем

Шаблон письма нужен для стандартизации оформления писем. Обычно шаблон включает в себя повторяющуюся информацию, а именно: название компании, ее логотип, фразу с приветствием, информацию о правах и так далее.


Система дает возможность отправлять письма в текстовом виде, а также в виде html-документа. Для переключения между режимами служат вкладки «Текстовое письмо» и «HTML-письмо», которые активируются нажатием левой кнопки мышки.




Важно! Если нужно отправить HTML-письмо, не забывайте переключаться в соответствующий режим, так как в режиме «Текстовое письмо» система автоматически убирает все html-теги из документа.


По умолчанию система включает стандартный шаблон, который может подойти для большинства писем. Если в выпадающем списке «Шаблон письма» ни один из шаблонов не выбран, то текст письма будет отправлен в том виде, в котором он был введен в поле «Текст письма». Это особенно важно, если письмо было отправлено в режиме «HTML-письмо».


Написание HTML-письма без указания шаблона рассчитано на то, что пользователь укажет html-разметку самостоятельно при помощи функционала редактора:




Для работы с html-разметкой необходимо обладать базовыми знаниями языка разметки гипертекста (HTML). При отсутствии знаний следует обратиться в техподдержку. О том, как это сделать можно прочесть в разделе «OneBox меню» → «Написать в техподдержку» или по этой ссылке .


При наличии знаний HTML можно вносить изменения в стандартный шаблон, который находится по пути: ProjectRoot/templates/default/mail/index.html. Также данный шаблон можно использовать как пример для создания собственных шаблонов.
Чтобы шаблоны начали отображаться в выпадающем списке «Шаблон письма», нужно скопировать их в папку: ProjectRoot/media/mail-design/. Важно! При создании шаблона не забудьте включить переменную [content] в свой шаблон: в нее система заносит текст письма.
