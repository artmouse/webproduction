Импорт и экспорт Excel

CRM система OneBox позволяет импортировать и экспортировать контакты в формат XLS. Чтобы воспользоваться данным функционалом необходимо активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Контакты».




На странице контактов нажмите на вкладку «Импорт и экспорт Excel».




Система выполнит переход к странице импорта и экспорта контактов, которая выглядит следующим образом:




Страница включает пошаговую инструкцию о том, как правильно пользоваться функционалом.
Выпадающий список позволяет выбрать, с какой группой контактов будет работать система. В поле «Send XLS files to emails» нужно указать email, на который придет сформированный файл XLS.
Для подтверждения изменений нажмите на кнопку «Скачать все данные в Excel».




Процесс загрузки контактов из XLS также интуитивно понятен.
Прикрепите файл XLS при помощи кнопки «Обзор...», затем нажмите на «Закачать все данные из Excel». После этого система добавит контакты в базу системы.
