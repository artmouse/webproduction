Редактирование контакта

Для редактирования контакта необходимо активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Контакты».




На странице всех контактов кликните по названию контакта, который нужно отредактировать. После этого система выполнит переход к карточке контакта. Выглядит она так:




Практически все поля в карточке являются редактируемыми.


Для удаления контакта следует воспользоваться пунктом «Удалить» в горизонтальном меню.
