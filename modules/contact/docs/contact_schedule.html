Рабочий график

В горизонтальном меню карточки контакта находится пункт «Рабочий график».




Функционал вкладки дает возможность ограничивать работу с системой для пользователей. Это возможно только при условии, что checkbox «Разрешать работать в системе только в заданное рабочее время», который находится во вкладке «Права доступа», активирован. Подробней о правах доступа можно прочесть в разделе «Контакты» → «Редактирование контакта» → «Права доступа» или по этой ссылке .


Важно! При активации checkbox`а задать время на рабочих графиках пользователей нужно обязательно, иначе они не смогут работать в системе.

По умолчанию она отображает семь дней недели, включая текущий и предыдущий день. Это поведение можно изменить при помощи панели фильтрации, которая находится в левой части экрана.
В панель включены два поля: «Дата с» и «Дата по» - они позволяют задать временной диапазон для построения таблицы.


Поля активируются нажатием левой кнопкой мышки. Чтобы изменения вступили в силу необходимо нажать на красную кнопку «Фильтровать», которая должна появиться чуть правее панели.




Рабочий график устанавливается при помощи checkbox`ов в таблице. Если checkbox отмечен — в системе можно работать, если нет - система недоступна для работы. К примеру, галочка установлена в checkbox`е напротив времени «09:00», это означает, что с 9:00 пользователь может приступать к работе. Напротив поля «13:00» галочки нет, так как начался обеденный перерыв — означает, что система кикнет всех пользователей с неотмеченным checkbox`ом. Это произойдет с любым очередным действием пользователя, который остался работать в системе после 13:00.
