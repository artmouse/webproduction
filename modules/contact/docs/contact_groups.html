Группы контактов

CRM система OneBox предоставляет возможность объединять контакты в группы, а также создавать группы для контактов вручную.

Для того, чтобы воспользоваться данным функционалом необходимо перейти в пункт меню «Настройки» → «Группы контактов».




Страница с группами контактов выглядит так:




Все группы, доступные по умолчанию, объединены в таблицу. Каждое поле таблицы отображает определенную информацию о группе.
Нажатие на заголовок таблицы позволяет сортировать записи в алфавитном порядке. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок.
Названия групп контактов являются ссылками. При нажатии на ссылку левой кнопкой мышки система выполнит переход к форме редактирования группы.


При наведении курсора на заголовок таблицы изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение — 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.


Чтобы добавить новую группу нужно воспользоваться вкладкой «Добавить группу».




Форма для добавления группы выглядит таким образом:




Среди полей формы можно отметить следующие:

  • «Название группы» — поле необходимое для корректного отображения группы в системе.
  • «Родительская группа» - если необходимо, чтобы новая группа была подгруппой.
  • «Описание группы» - текстовое описание для лучшего понимания, какие контакты должны быть включены в группу.
  • «Уровень цен» - группе контактов можно назначить индивидуальный уровень цен продуктов.
  • «Сортировка» - определяет положение группы в панели фильтрации, которая находится на странице отображения всех контактов. Если установлен 0 - то сортировка происходит в алфавитном порядке, если любое другое число, то чем ближе оно к 1, тем выше у группы приоритет при сортировке.
  • «Цвет» - можно указать html-кодом, к примеру, #ffffff. Фон для таких контактов в общем списке контактов изменится.

Для создания группы необходимо нажать на кнопку «Вставить новую запись», которая находится в нижней части экрана.


Если группа уже была создана и в нее нужно внести коррективы или удалить, кликните на название группы в общем списке. На экране появится форма для редактирования группы:




Форма обладает аналогичным набором полей, что и форма для добавления группы. Все поля для настройки сгруппированы в блоке слева.
Из отличий можно отметить опцию «Добавить поле», которая находится в блоке справа. Она позволяет добавлять в карточку контакта, который относится к определенной группе, новые поля. Созданные поля не будут отображаться в контактах вне группы. Работа данного функционала описана в разделе «Контакты» → «Поля контактов», прочесть о нем можно, перейдя по этой ссылке .

Для сохранения изменений нажмите на кнопку «Сохранить», которая находится внизу экрана.
Для удаления группы воспользуйтесь кнопкой «Удалить». На экране появится модальное окно с просьбой подтвердить удаление записи. Важно! Контакты, которые состоят в группе удалены не будут.


Чтобы посмотреть, к какой группе принадлежит контакт зайдите в его карточку. Более подробно об этом можно прочесть в разделе «Контакты» → «Редактирование контакта» → «Общая информация о контакте» или по этой ссылке .
