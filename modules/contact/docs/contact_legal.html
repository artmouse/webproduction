Юридические реквизиты контакта

В горизонтальном меню карточки контакта находится пункт «Юридические реквизиты».




Данный функционал позволяет добавить юридические реквизиты, которые будут автоматически подтягиваться системой при формировании документов.


Страница добавления реквизитов выглядит следующим образом:




Все ранее созданные реквизиты отображены в виде таблицы.
Выпадающий список «Добавить юридические реквизиты» служит для создания формы реквизитов. Учтите, что поля формы меняются в зависимости от выбранной страны.
Для заполнения поля нужно активировать его нажатием левой кнопкой мышки.


Для того, чтобы система сохранила внесенные изменения следует нажать на кнопку «Добавить»:




После этого произойдет перезагрузка страницы и в списке всех реквизитов появится новый реквизит.


Удалить реквизит можно в таблице всех реквизитов. Для этого нужно нажать на ссылку «Удалить»:




Далее система попросит подтверждение о выполнении данного действия.
