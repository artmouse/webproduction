Просмотр документов

Чтобы просмотреть все созданные документы, необходимо активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Документы».




Страница документов выглядит следующим образом:




По умолчанию система отображает все документы без использования каких-либо фильтраций. Подтверждением этого является активная закладка «Все документы» в верхней части экрана:




Документы можно фильтровать при помощи других закладок:


Входящий документ - тот который пришел нам (счет-фактура на закупку от поставщика)
Исходящий - от нас (счет-фактура клиенту заказа)
Внутренний - к примеру, штрих-код товара




Заголовки колонок используются для сортировки записей. Для этого нужно нажать на заголовок левой кнопкой мышки. Выполнить обратную сортировку можно повторным нажатием на заголовок.




При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.


Множество полей в списке документов являются ссылками. Это значит, что при нажатии, к примеру, на имя автора документа система выполнит переход к карточке контакта, где можно будет более детально изучить персональные данные автора.
Для того, чтобы внести изменения в документ нужно кликнуть на его название или номер. После этого на экране появится форма для редактирования документа.




В списке документов предусмотрено удаление. Для этого нужно нажать на ссылку «Удалить» напротив определенного документа. После этого появится модальное окно, в котором нужно будет подтвердить удаление записи.




Также предусмотрена возможность распечатки документа. За это отвечает кнопка «Печать».




В самом низу списка находится функционал для экспорта данных в форматы CSV, XLS и XML. За это отвечает ссылка «Экспорт в».
Также здесь находится пагинатор, который позволяет листать документы постранично. Для этого следует выбрать номер страницы или воспользоваться кнопками «Назад» и «Вперед» для перехода на предыдущую/следующую страницу соответственно.
