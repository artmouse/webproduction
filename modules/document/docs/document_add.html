Добавление документов

Процесс добавления документов непосредственно связан с процессом создания шаблонов для документов. В CRM системе OneBox формирование документов происходит по шаблонам. Используя шаблон, сформированный в формате HTML-разметки, можно создать практически любой документ.


По умолчанию система предоставляет несколько шаблонов на трех языках (EN, RU, UA): счет-фактура, накладная заказа и акт выполненных работ.
Также можно использовать свои собственные шаблоны, однако для этого нужно обладать знаниями в области HTML-верстки.


Ознакомиться с процессом добавления новых шаблонов можно по этой ссылке .


Процесс добавления документа является связанным либо по отношению к проекту или заказу, либо по отношению к контакту.
Привязка документа происходит в форме добавления шаблона, за это отвечает опция «К чему относится?». Поэтому, если документ закреплен за контактами, то его нельзя будет создать в контексте заказа или проекта.




Пример добавления документа можно посмотреть в разделе «Заказы» или по этой ссылке .
