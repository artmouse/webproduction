Шаблоны документов

CRM система OneBox позволяет создавать и редактировать шаблоны документов, на основе которых будут автоматически генерироваться документы к заказам, проектам и контактам.


Чтобы начать работу с шаблонами нужно активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Настройки».




После этого найдите и кликните на пункт «Шаблоны документов»:




Страница управления шаблонами будет выглядеть так:




По умолчанию в системе созданы шаблоны документов на трех языках (EN, RU, UA): счет-фактура, накладная заказа, акт выполненных работ.

На странице шаблонов документов доступны следующие действия:


Быстрое управление шаблонами — удобный функционал для внесения быстрых поправок. К примеру, при нажатии на ячейку «Скрытый» появится checkbox, значение которого можно менять прямо с текущей страницы.

После этого появится форма для добавления шаблона:




Важно! Шаблон документа изначально должен быть сформирован в формате HTML-разметки, где будут корректно указанны все размеры и отступы. Только в этом случае им можно пользоваться и добавлять в систему. Также это дает возможность использовать специальные переменные в контексте документа.


Форма для добавления шаблона содержит следующие поля:

  • Название документа - необходимо указать, как будет называться шаблон документа в системе (к примеру "Накладная заказа")
  • К чему относится? - здесь необходимо выбрать, к чему относится данный документ. Если выбрано «К проектам и заказам», то формировать документ можно будет только в контексте проекта или заказа. Если выбрано «К контактам» - то только в контакте
  • Скрытый - если проставить галочку, то документ будет недоступен для формирования
  • Обязательный документ - если включить данную опцию, то документ будет автоматически сформирован при создании заказа
  • Срок действия документа в днях - числовое значение, по истечению которого завершится срок действия документа
  • Порядок сортировки - положение документа в общем списке при добавлении документа к заказу/проекту (если не указать, то будет в алфавитном порядке)
  • Программный обработчик нумерации - используется для того, чтобы можно было предложить разработчикам свой алгоритм формирования номера документа (по умолчанию номер формируется по порядку начиная от 1)
  • Шаблон документа - в данное поле необходимо вставлять шаблон в формате HTML-кода. В блоке «Отображение шаблона» можно увидеть результат внешнего вида документа

После заполнения всех полей необходимо нажать на кнопку «Добавить запись». Таким образом шаблон будет сохранен в системе.


Редактирование шаблона документа

Для внесения изменений в шаблон документа нужно нажать на «Название документа» в списке всех шаблонов. После этого система выполнит переход к форме редактирования шаблона, которая обладает аналогичным набором полей, что и форма для добавления шаблона, описанная выше.


Обратите внимание, что форма дает возможность вручную редактировать код шаблона документа (вкладка «Редактор»). Однако, для этого нужно обладать знаниями в области HTML-верстки, в противном случае лучше подать заявку в службу технической поддержки. Чтобы это сделать необходимо активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Написать в техподдержку».


После внесения изменений нажмите кнопку «Сохранить запись».
