Панель фильтрации

Списком можно управлять при помощи панели фильтрации. Это удобно для быстрого поиска конкретного документа среди большого количества наименований. Активируется панель в левой части экрана нажатием на специальную вкладку, как указано на рисунке:




После этого экран будет выглядеть так:




Все поля панели являются критериями для фильтрации списка и активируются путем нажатия на них левой кнопкой мышки. Для того, чтобы изменения вступили в силу следует нажать на кнопку «Фильтровать», которая появится чуть правей панели.




Если в результате фильтрации список документов не был сформирован — это значит, что документов соответствующих заданным критериям в базе не существует.


Панель фильтрации состоит из следующих полей:


«Номер» - соответствует id документа.
«Название» - названию документа.
«Направление» - дублирует вкладки в верхней части экрана.
«Подгруппа» - отвечает за выбранный шаблон документа.
«Автор» и «Клиент» - фильтровать документы по автору и клиенту.
«Заказ» - отфильтровать документы по номеру заказа, к которому они относятся.
Статусы документа — позволяют выполнять фильтрации ориентируясь на текущий статус документа:

  • сформирован
  • отправлен
  • получен
  • в архиве

В самом низу панели находятся поля, отвечающие за фильтрацию по дате. С их помощью можно задать диапазон для поиска нужного документа.
