Файлы

Таблица файлов

В горизонтальном меню карточки заказа существует вкладка «Файлы». Вкладка дает возможность просмотреть все вложенные файлы в контексте заказа. Это удобно, когда среди большого количества информации в карточке заказа нужно найти вложенный файл и скачать его.


Перейти к содержимому вкладки можно, кликнув на нее левой кнопкой мышки.




Страница файлов выглядит следующим образом:




Все файлы, которые были прикреплены к заказу отображены в виде таблицы. Здесь можно увидеть кем, когда и какой файл был добавлен в заказ.
Некоторые поля списка являются ссылками. «Имя файла» или номер файла ведут к странице просмотра содержимого файла. Имя автора — к карточке контакта, поле «Скачать» позволяет приступить к скачиванию контента.




Таблица позволяет сортировать данные. Происходит сортировка по нажатию на заголовок колонки. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок.
Под таблицей находится функционал для экспорта документа в форматы: CSV, XLS, XML. Это реализовано с помощью меню, которое активируется по клику мышкой.


При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.


Как уже было сказано, при нажатии на имя или номер файла система выполнит переход к странице просмотра содержимого файла.




К примеру, если файл является картинкой, то на экране будет отображено название картинки, а под ним сама картинка.
Ниже находится ссылка для скачивания: «Скачать файл».
Checkbox «Удалить файл» - если контент нужно удалить из заказа, отметьте поле галочкой и нажмите на кнопку «Сохранить».
«Заменить файл» - поле для выбора нового контента, который заменит уже имеющийся. Замена произойдет также после нажатия на кнопку «Сохранить».




Панель фильтрации

Таблицу файлов можно фильтровать при помощи панели фильтрации. Активируется она в левой части экрана путем нажатия на специальную вкладку, как указано на рисунке:




После этого экран будет выглядеть так:




Все поля панели являются критериями для фильтрации и активируются путем нажатия на них левой кнопкой мышки. Для того, чтобы изменения вступили в силу следует нажать на кнопку «Фильтровать», которая появится чуть правей панели.




Если в результате фильтрации список файлов не был сформирован — это значит, что файлов, соответствующих заданным критериям, в базе не существует.


Панель фильтрации состоит из следующих полей:
«Имя файла» - фильтрация по имени файла.
«Привязка» - отображает номер заказа, к которому привязаны файлы.
«Автор» - фильтрация данных по имени менеджера, который ответственный за вложение контента в заказ.
«Дата загрузки от» и «Дата загрузки до» - позволяет задать временный диапазон для поиска файла.
«ID» - фильтрация по номеру файла.
«Тип» - фильтрация по типу файла.
