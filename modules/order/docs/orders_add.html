Добавление нового заказа

Для добавления нового заказа нужно перейти в раздел «Заказы», затем воспользоваться вкладкой «Добавить новый заказ», которая находится в верхней части экрана:




Форма добавления заказа разделена на два блока. Левый блок служит для поиска продуктов, из которых формируется заказ.


Поле «Категория» представлена выпадающим списком. В поле «Список продуктов» нужно ввести название продукта. Для того, чтобы начать поиск нужно кликнуть на кнопку «Найти».
Система пытается показать все возможные варианты записей на случай, если пользователь не знает точного названия продукта.
Если список продуктов не был сформирован, нужно еще раз проверить корректность заполнения поля «Список продуктов» или попробовать искать продукт по другому названию.




Результатом поиска является список товаров в блоке поиска продуктов. Если кликнуть левой кнопкой мышки по товару, он автоматически добавится в корзину, как на рисунке ниже:




Также в левом блоке формы находится пункт «Дополнительные настройки». По умолчанию он скрыт. Чтобы его развернуть нужно нажать на него левой кнопкой мышки.


После открытия блок выглядит таким образом:




Все поля дополнительных настроек являются необязательными для формирования заказа системой. Они несут в себе уточняющий характер, который немаловажен для ответственных по проекту или заказу.


Перечень полей заказа выглядит так:

  • Когда будет старт заказа
  • Название заказа
  • Описание заказа
  • К какому проекту относится
  • Канал заказа
  • Бизнес-процесс
  • В каком виде должен быть результат
  • Оценочное время
  • Оценочные деньги
  • Прочие ресурсы

В правом блоке формы находится корзина, несколько уточняющих полей, а также список для формирования документов.


Товары в корзине можно редактировать. К примеру, можно изменить цену на товар или количество товаров в заказе.
При любых правках в корзине цена и общая стоимость заказа автоматически пересчитывается системой.
Для удаления товара из корзины нужно нажать на красную кнопку «Х».




Также при добавлении заказа есть возможность сформировать по нему документы. Для этого нужно отметить checkbox`ы документов из списка «Сформировать документы»:




Для сохранения всех изменений следует нажать кнопку "Сохранить".
