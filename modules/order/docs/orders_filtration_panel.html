Панель фильтрации заказов

Панель фильтрации доступна во всех режимах отображения заказов. Активируется она в левой части экрана путем нажатия на специальную вкладку, как указано на рисунке:




В открывшейся панели фильтрации вы можете:

  1. Изменить способ отображения. Более подробно о способах отображения можно почитать в разделе «Календарь», либо по этой ссылке .

  2. Выбрать ответственного:


  3. Фильтровать по клиентам, названиям заказов, дате создания(от, до) .
    Поле «Код» соответствует id заказа, который отображается как колонка с заголовком «#».


  4. Также есть возможность фильтровать заказы по авторам и регулировать отображение закрытых или выполненных заказов.


  5. В этом же блоке находятся бизнес-процессы, по которым можно выполнять фильтрацию. Для этого нужно отметить галочкой определенный бизнес-процесс и нажать кнопку «Фильтровать».
    Каждый бизнес-процесс состоит из этапов. Посмотреть их можно, нажав на бизнес-процесс левой кнопкой мышки.
    Поля этапов в выпадающем списке бизнес-процесса также представлены checkbox`ами, которые аналогично используются в качестве критериев для фильтрации.
