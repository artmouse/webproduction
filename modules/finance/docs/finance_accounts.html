Финансы: кошельки и счета

Список всех кошельков

Для корректной работы с платежами в систему необходимо добавить счета, по которым будут проходить транзакции. Счета в рамках системы называются аккаунтами. Настраивается этот функционал в настройках.


Чтобы просмотреть все кошельки доступные на данный момент в системе, нужно активировать меню OneBox, которое находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Настройки».




Далее в открывшемся списке нужно найти пункт «Финансы: категории и фонды» и кликнуть по нему.




Счета объединены в таблицу, которая позволяет сортировать данные. Происходит сортировка по нажатию на заголовок колонки. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок.
Под таблицей находится функционал для экспорта документа в форматы: CSV, XLS, XML. Это реализовано с помощью меню, которое активируется по клику мышкой.
Здесь же находится пагинатор, который позволяет листать наименования постранично. Для этого следует выбрать номер страницы или воспользоваться кнопками «Назад» и «Вперед» для перехода на предыдущую/следующую страницу соответственно.




При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.


Справа от таблицы расположен блок с информацией о фондах, аккаунтах и курсом валют.
Фонды — по сути категории, которые создаются для объединения платежей. Настройка фондов происходит в «Настройки» → «Финансы: категории и фонды», более подробно о данном функционале можно прочесть в соответствующем разделе или по этой ссылке .
Курсы валют — отображение курса по отношению к базовой валюте. Эти параметры можно задавать вручную, также система позволяет установить для них автообновление. Настройка функционала происходит в OneBox меню: «Настройки» → «Валюты».


Названия аккаунтов в таблице являются ссылками, при нажатии на которые можно перейти к редактированию счета. Форма для редактирования обладает аналогичным набором полей, что и форма для добавления счета.


Добавить новый кошелек

Для добавления новой категории нужно нажать на вкладку «Добавить новый кошелек», которая находится в верхней части экрана.




Форма для добавления выглядит следующим образом:




«Название кошелька» - поле обязательно для заполнения. Если оставить его пустым система не сможет создать новую запись и выдаст ошибку.
«Описание кошелька» - краткое описание счета, может содержать, к примеру, пометку для каких целей данный аккаунт используется.
«Валюта» - выпадающий список со всеми валютами, указанными в системе. Настройка функционала происходит в OneBox меню: «Настройки» → «Валюты».
Checkbox «Активен» - служит для временной деактивации счета.
«Юридическое лицо» - выбор юридического лица.
«Стартовый баланс» - сумма на счету.


После того, как поля были заполнены, нажмите на кнопку «Создать аккаунт» для сохранения изменений.
