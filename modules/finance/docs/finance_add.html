Добавление платежа

Для добавления нового платежа нужно активировать меню OneBox, которое находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Финансы».




На странице всех платежей кликните по вкладке «Новый платеж», которая находится в верхней части экрана:




Произойдет переход к форме для добавления нового платежа. Выглядит она так:




Поле «На сумму» - обязательно для заполнения. Также, для добавления платежа важно указать клиента, иначе система не будет знать, кому адресован данный платеж.


«Клиент» - поле для ввода имени клиента. Начните вводить имя и система подскажет, какие имена клиентов уже присутствуют в системе. Для того, чтобы добавить имя — нажмите на него левой кнопкой мышки. В поле может быть только одно имя клиента.
Также для добавления клиента можно воспользоваться кнопкой «Выбрать контакт».




Это приведет к переходу к следующей форме:




Знак «Х» в правом верхнем углу позволяет вернуться на предыдущую страницу.
Поле «Найти клиента» - нужно для ввода имени клиента. После того, как имя было указано нажмите на кнопку «Найти», все найденные наименования будут отображены в списке под полем. Кликните на имя клиента, чтобы добавить его в платеж, система при этом вернется к предыдущей форме.


Также форма для поиска клиентов позволяет создавать новых клиентов. Для этого нужно воспользоваться вкладкой «Создать клиента»:




Форма для создания выглядит так:




Она позволяет создавать как физические, так и юридические лица. Поле «Имя» - является обязательным для ввода, если оставить его пустым система выдаст ошибку.
Все остальные поля — уточняющие.
Для того, чтобы завершить создание контакта, нажмите на кнопку «Создать». После этого можно будет завершить создание платежа.


В форме для добавления платежа также присутствуют и другие поля.
«Счет» - представлен выпадающим списком, который формируется из списка всех аккаунтов в настройках.
«Номер заказа» - позволяет закрепить платеж за заказом.
«Категория оплаты» - поле для группировки расходов, за отображение которых отвечает функционал модуля «Finance».
Для указания банковских реквизитов нажмите на ссылку «Банковские реквизиты». Ссылка открывает два новых поля: «Референс код» и «Банковские реквизиты».




Кроме того форма дает возможность прикрепить файлы, для этого следует воспользоваться полем «Файл».


После внесения всех данных, для подтверждения изменений, нажмите на кнопку «Сохранить», которая находится внизу формы. Если нужно добавить несколько платежей подряд - используйте кнопку «Сохранить и добавить еще».
Для возвращения к списку всех платежей кликните по вкладке «Все платежи».
