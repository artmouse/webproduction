Панель фильтрации

На странице всех платежей присутствует панель фильтрации. Она предназначена для фильтрации таблицы по различными критериям. Активируется панель в левой части экрана нажатием на специальную вкладку, как указано на рисунке:




После этого экран будет выглядеть так:




Все поля панели являются критериями для фильтрации и активируются путем нажатия на них левой кнопкой мышки. Для того, чтобы изменения вступили в силу следует нажать на кнопку «Фильтровать», которая появится чуть правей панели.




Если в результате фильтрации список событий не был сформирован — это значит, что платежей, соответствующих заданным критериям, в базе не существует.


«Юридическое лицо» - фильтрация по юридическим лицам.
«Счет» - выпадающий список для выбора аккаунта.
Поле для сортировки по статусу счета: проведен, создан.
«Дата с» и «Дата по» - позволяют задать временный промежуток, в рамках которого происходил платеж.
Поле «Комментарий» - дает возможность искать платеж по ключевым словам в комментариях, оставленных в платежах в поле «Комментарий».
Также фильтрация таблицы может происходить по клиентам, менеджерам, категориям и направлению платежа: все, входящие, исходящие.
