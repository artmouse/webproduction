Исполнители

В горизонтальном меню карточки проекта существует вкладка «Исполнители». Она отображает всех исполнителей проекта, а также дает возможность управлять ими. Этот функционал полезен, когда за разными этапами проекта нужно закрепить разных исполнителей.


Перейти к содержимому вкладки можно, кликнув на нее левой кнопкой мышки.




Страница «Исполнители» выглядит следующим образом:




Все исполнители отображены в таблице. Над таблицей указан номер текущего проекта, сумма заказов в рамках проекта, а также сумма, которая была распределена между исполнителями.


В таблице каждая строка соответствует одном исполнителю. При наведении мышкой на строку — она подсвечивается.
Поле «Исполнитель» является выпадающим списком, который формируется из структуры предприятия. О том, как работать со структурой в системе OneBox и зачем она нужна, можно прочитать в разделе «Структура» или по этой ссылке .


Поле «Этап» позволяет назначать исполнителя для конкретного этапа.
«Крайний срок этапа» - задать сроки для выполнения этапа.
Поле «Сумма» служит для распределения средств между исполнителями.


После того, как все поля будут заполнены, нажмите на кнопку «Сохранить» для подтверждения изменений.
