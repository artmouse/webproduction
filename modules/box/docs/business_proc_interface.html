Настройка этапов бизнес-процесса: интерфейс

Интерфейс — позволяет сформировать главную страницу заказа, проекта или задачи. Такая страница обязательно разделена на блоки, а каждый блок отвечает за свой функционал. К примеру, главная страница проекта выглядит так:




С помощью интерфейса можно кастомизировать страницу практически на любой вкус: добавить новые блоки и удалить ненужные.


Важно! Данный функционал будет работать только в том случае, если в главном конфигурационном файле engine.mode включена опция «показывать в меню типы workflow».


Страница интерфейса выглядит следующим образом:




Слева в списке находятся блоки, из которых можно собирать страницу. Их можно перемещать, используя метод drag-and-drop. Для этого нужно зажать левую кнопку мышки на нужном блоке, затем навести курсор на один из желтых блоков.
На странице также предусмотрен быстрый поиск блока. Введите в поле название блока и список будет отсортирован.


Желтые блоки - это разметка главной страницы заказа, проекта или задачи. Такая структура позволяет наглядно видеть страницу, которая будет сформирована.


Для того, чтобы система запомнила изменения, следует нажать на кнопку «Сохранить». После вышеописанных действий страница заказа, проекта или задачи должна поменяться.
