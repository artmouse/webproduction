Добавление задачи

Чтобы добавить задачу, нужно навести указатель мышки на меню OneBox. Далее выбрать пункт «Задачи».




В горизонтальном меню, в верхней части экрана, нажмите на закладку «Создать задачу», как на рисунке:




После выполнения вышеописанных действий должна появиться форма для создания задачи.
"Название" и "бизнес-процесс" - поля, которые нужно заполнять обязательно, иначе создать задачу не удастся.
Все остальные поля являются необязательными для заполнения, однако несут в себе уточняющий характер, который немаловажен для ответственного по задаче.


Проект — поле для выбора проекта в рамках, которого будет создана задача. Начните вводить в поле название проекта и система сама подскажет, какие проекты доступны на текущий момент.
Ответственный — выбор персоны для выполнения задачи.
Описание — описание того, что нужно выполнить в рамках поставленной задачи.
Также в задачу можно вкладывать файлы при помощи поля «Приложить файлы».


После того как все поля формы были заполнены, завершить процесс создания задачи нужно нажатием на кнопку «Сохранить» в нижней части экрана.




Для того, чтобы после создания задачи система выполнила перенаправление на создание новой задачи, следует воспользоваться кнопкой «Сохранить и добавить еще».


Добавлять задачи в OneBox можно и другими способами. Один из самых простых и удобных — добавление задачи через календарь, о котором можно почитать в разделе «Календарь» → «Добавление задач и заказов через календарь» или по этой ссылке .


Также имеется возможность создавать задачи в контексте проекта. Об этом можно почитать в разделе «Задачи» → «Добавление задач из проектов» или по этой ссылке .
