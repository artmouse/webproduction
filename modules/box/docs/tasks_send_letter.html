Написать письмо из задачи

Если в задаче указан клиент, а у клиента в персональной информации указана электронная почта, то в блоке для написания комментариев можно увидеть вкладку «Написать письмо». Она позволяет перейти к форме для отправки писем. Для перехода нажмите на вкладку левой кнопкой мышки, после чего блок будет выглядеть так:




Поле «Введите E-mail» нужно для вставки электронной почты, как правило, оно заполняется автоматически.
Далее идут поля для указания темы и тела письма.
Ниже — адрес отправителя.
Также в письмо можно вложить файлы — за это отвечает кнопка «Приложить файлы».


После того, как все поля были заполнены, для отправки письма нажмите на кнопку «Сохранить», которая находится в нижней части экрана.


Важно! В случае, если в главном конфигурационном файле включена опция «Показывать в меню типы workflow», то в форме будет отображаться checkbox «Отправить красивое HTML-письмо». Это дает возможность писать письма в поле «Сообщение» в формате HTML-разметки. Для использования данного функционала необходимы базовые знания по HTML. В случае затруднений с этим вопросом можно обратиться в техподдержку. О том как это сделать, можно прочесть в разделе «OneBox меню» → «Написать в техподдержку» или по этой ссылке .
