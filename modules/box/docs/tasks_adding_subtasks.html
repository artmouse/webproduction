Создание подзадач

CRM система OneBox дает возможность создавать задачи в контексте других задач. В этом случае задача, в рамках которой была поставлена подзадача, становится для нее проектом.


Важно! Задачи не будут создаваться, если в системе нет бизнес-процесса, который установлен для новых задач. В этом случае система может НЕ выдать ошибку, однако новые задачи тоже НЕ появятся.


Чтобы воспользоваться функционалом, нужно выбрать задачу в списке всех задач.




Затем, среди полей задачи, нужно найти блок для написания комментариев. Как правило, он находится под информационным блоком:




Блок комментариев по умолчанию находится не только в задачах, но и в проектах, и заказах. Процесс создания подзадач из проектов и заказов будет выглядеть аналогично, с той разницей, что задачи будут создаваться в контексте родительского проекта/заказа.


Для того, чтобы поставить подзадачу, переключитесь на вкладку «Поставить задачу» левой кнопкой мышки. Вкладка станет активной:




Теперь в поле для ввода текста нужно ввести символ «@». Появится выпадающий список сотрудников:




Для добавления сотрудника кликните на его имя — это позволяет установить ответственного для задачи. Дальнейший ввод текста задает заголовок задачи: все символы будут расцениваться системой как название заголовка.
Нажатие на клавишу «enter» сдвинет курсор на новую строку, а также даст возможность перейти к указанию описания задачи. Ввод будет выглядеть таким образом:




Если в поле ввода пропустить одну строку, то следующие введенные символы будут восприняты системой как еще один заголовок задачи. Таким образом можно создавать для одного сотрудника несколько задач с разными названиями и описаниями.

После внесения изменений нажмите на кнопку «Сохранить», которая находится внизу экрана. Новые задачи появятся в списке всех задач.


Обратите внимание на поле «Проект» в информационном блоке подзадачи. В нем будет указан код и название задачи, в рамках которой ставилась подзадача.
