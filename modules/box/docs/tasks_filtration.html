Панель фильтрации задач

Панель фильтрации доступна во всех режимах отображения задач. Активируется она в левой части экрана путем нажатия на специальную вкладку, как указано на рисунке:




После активации панель фильтрации будет выглядеть так:




Одной из ключевых возможностей панели является управление способами отображения.
Более подробно о способах отображения можно почитать в разделе «Календарь», либо по этой ссылке .

Помимо выбора режимов отображения панель дает возможность фильтровать задачи по всевозможным критериям.
Все поля активируются нажатием на них левой кнопкой мышки, а чтобы изменения вступили в силу, необходимо нажать на красную кнопку «Фильтровать», которая должна появиться чуть правее панели.




В панели предусмотрена фильтрация по имени ответственного сотрудника, а также по клиентам и авторам.




Поле «Название» соответствует колонке «Имя задачи».
Поле «Код» соответствует id задачи, который отображается, как колонка с заголовком «#».
Дата создания от и до - позволяет задать временный промежуток для поиска нужной задачи.




По умолчанию список задач включает отображение закрытых или выполненных задач. Это можно изменить, убрав галочку в checkbox`е «Показывать закрытые».
В этом же блоке находятся бизнес-процессы, по которым можно выполнять фильтрацию. Для этого нужно отметить галочкой определенный бизнес-процесс и нажать кнопку «Фильтровать».
Каждый бизнес-процесс состоит из этапов. Посмотреть их можно, нажав на бизнес-процесс левой кнопкой мышки.
Поля этапов в выпадающем списке бизнес-процесса также представлены checkbox`ами, которые аналогично используются в качестве критериев для фильтрации.
