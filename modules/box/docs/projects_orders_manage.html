Управление заказами проекта

В рамках одного проекта можно создавать, просматривать и редактировать заказы. Для управления заказами из проекта следует активировать выдвижную панель, которая находится в левой части экрана. Затем нажать левой кнопкой мышки на пункт «Проекты».




В появившемся списке нужно выбрать один из проектов. Для этого нужно кликнуть на его название или номер, как показано на рисунке:




После этого на экране появится форма для редактирования проекта. В верхней части формы, в горизонтальном меню, есть пункт «Заказы», нажмите на него.




Для добавления заказа следует воспользоваться кнопкой «Создать заказ».
