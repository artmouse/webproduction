Панель быстрого управления задачами

При просмотре всех задач существует возможность вносить изменения, используя панель быстрого редактирования. Чтобы воспользоваться этим функционалом нужно активировать выдвижную панель, которая находится в левой части экрана. Затем нажать левой кнопкой мышки на пункт «Задачи».




По умолчанию панель быстрого редактирования скрыта. Для ее активации нужно отметить нужный проект галочкой, как показано на рисунке ниже:

Панель быстрого редактирования позволяет выполнять следующие действия:

  • сменить ответственного
  • поменять статус задачи/заказа
  • установить дату завершения
  • закрыть задачу/заказ
  • открыть задачу/заказ
  • удалить задачу/заказ

Все поля панели активируются путем нажатия на них левой кнопкой мышки, а для того, чтобы изменения вступили в силу, следует нажать на кнопку «Изменить».
При помощи поля «Открыть задачи» можно открыть ранее закрытые задачи.
Особенно осторожно следует пользоваться кнопкой «Удалить», так как на данный момент возможность восстановить задачу в системе не предусмотрена.


Массовая отправка комментариев

В случае, если в главном конфигурационном файле системы включена опция «Показывать в меню типы workflow», в панели быстрого управления будет доступен дополнительный функционал для массовой отправки комментариев. Это удобно, если один и тот же комментарий нужно оставить в нескольких задачах, заказах, проектах.




Форма отправки включает в себя поле для ввода сообщения и кнопку «Добавить», которая отвечает за отправку комментария. Установите курсор мышки в поле, введите сообщение, затем нажмите «Добавить». Новый комментарий появится во всех задачах, которые были отмечены в checkbox`ах.
