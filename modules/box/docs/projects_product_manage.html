Управление продуктами проекта

В рамках одного проекта можно создавать, просматривать и редактировать продукты. Для управления продуктами из проекта следует активировать выдвижную панель, которая находится в левой части экрана. Затем нажать левой кнопкой мышки на пункт «Проекты».




В появившемся списке нужно выбрать один из проектов. Для этого нужно кликнуть на его название или номер, как показано на рисунке:




После этого на экране появится форма для редактирования проекта. В верхней части формы, в горизонтальном меню, есть пункт «Продукты», нажмите на него.




Для того, чтобы найти и добавить продукт в проект нужно ввести его название в поле ввода рядом с кнопкой «Выбрать/найти/создать продукт»:




В выпадающем списке будут отображены все возможные варианты продуктов, которые на данный момент существуют в базе. Если выпадающий список не появился, значит такого продукта не существует.
Для выбора продукта нужно кликнуть по его названию. Выбранный продукт будет отображен на экране сразу над полем для ввода. Таким образом, можно выбирать наименования поочередно, добавляя их в проект и формируя новый список.




Для того, чтобы отредактировать продукт, нужно воспользоваться полем «Редактировать все». Находится оно с правой стороны, над стоимостью. При нажатии на поле, оно сменится надписью «Отменить все», которая нужна для отмены предыдущего действия.




После внесения изменений в продукт нужно подтвердить их нажатием на кнопку «Сохранить».




Порядок сформированного списка продуктов можно легко изменить. Для этого достаточно потянуть за корешок нужного продукта, как указано на рисунке:




Для удаления продукта из проекта следует нажать на красную кнопку с крестиком, которая находится справа от отображения цены продукта. Фон продукта при этом изменится на серый. Подтверждать удаление нужно обязательно, для чего существует кнопка «Сохранить»:




Важно понимать, что при удалении продукт удаляется из текущего проекта, а не из базы данных.


Для просмотра списка всех продуктов нужно нажать на кнопку «Выбрать/найти/создать продукт...».




Здесь отображается информация о номере, названии, наличии и стоимости продукта.
В отличии от быстрого поиска, данная страница позволяет фильтровать поиск по категории. Это полезно в том случае, если название продукта известно неточно или не известно вообще.
В остальном поиск происходит аналогично. В поле «Найти продукт» нужно указать точное или приблизительное имя продукта, после чего нажать на кнопку «Найти»:




Для выбора продукта нужно нажать на него левой кнопкой мышки. После этого система автоматически выполнит переход на предыдущую страницу, курсор мышки установится в поле поиска продукта рядом с его номером. Теперь для добавления продукта в список нужно нажать на кнопку «Сохранить».




На странице просмотра всех продуктов также есть возможность создавать новые продукты. Для этого вместо кнопки «Найти продукт» нужно выбрать кнопку "Создать продукт". После чего на экране появится форма создания нового продукта.




«Название продукта» является единственным обязательным полем для заполнения. Если продукт не имеет названия - система не сможет его создать, а на экране появится модальное окно с надписью name.
Все остальные поля являются уточняющими.
После заполнения всех полей нажмите на кнопку «Создать».




Дальше система автоматически выполнит переход на предыдущую страницу, курсор мышки установится в поле поиска продукта рядом с его номером. Теперь, для добавления продукта в список, нужно нажать на кнопку «Сохранить».
