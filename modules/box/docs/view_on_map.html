Способ отображения «На карте»

Чтобы выбрать способ отображения «На карте», необходимо активировать панель фильтрации в левой части экрана и поменять способ отображения на «На карте».




После выполнения вышеописанных действий экран будет выглядеть так:




Данный способ отображения позволяет увидеть местонахождение клиентов на картах «Яндекс.Карты». Адрес клиента отмечен специальным знаком, при нажатии на который появится сообщение в виде облака:




Способ отображения «На карте» используется для фильтрации задач, заказов и проектов. Каждый клиент, указанный в задаче, заказе или проекте, будет отмечен на карте. Если адрес клиента не указан, то на картах он отображаться тоже не будет.
Важно! Список задач, заказов и проектов разбит на страницы. Одновременно на карте отображаются клиенты только с текущей страницы.


Это удобно в сочетании с другими фильтрами из панели. К примеру, при выборе бизнес-процесса «Встреча» можно будет узнать, по какому адресу она состоится, а также посмотреть план города, чтобы иметь представление, куда конкретно нужно ехать.
