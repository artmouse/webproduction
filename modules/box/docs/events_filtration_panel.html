Панель фильтрации

Также на странице всех событий присутствует панель фильтрации. Она предназначена для быстрого поиска конкретного события или группы событий. Активируется панель в левой части экрана нажатием на специальную вкладку, как указано на рисунке:




После этого экран будет выглядеть так:




Все поля панели являются критериями для фильтрации и активируются путем нажатия на них левой кнопкой мышки. Для того, чтобы изменения вступили в силу, следует нажать на кнопку «Фильтровать», которая появится чуть правей панели.




Если в результате фильтрации список событий не был сформирован — значит событий, соответствующих заданным критериям, в базе не существует.


Панель позволяет фильтровать события по типу: Email, звонки, SMS. За это отвечает выпадающий список «Тип события».
«Направление события»: входящие, исходящие, внутренние. Входящие и исходящие — в зависимости от направления события, к примеру, если письмо отправлено из системы — исходящий тип события. Внутренние (наименования, прописанные в парсере) — когда используется внутренняя телефония, звонок с номера 405 на 402 или когда письмо отправлено внутри системы, с user1@webproduction.ua на user2@webproduction.ua.
«Каналы» - номера телефонов/симок, на которые попадает звонок при интеграции с телефонией.
«Статус (для звонков)» - фильтрация звонков по статусу звонка: разговор состоялся, не взяли трубку, сбросил и т.д.
Поля «Дата от» и «Дата до» - позволяют задать временной промежуток для поиска событий.
«От кого» и «Кому» - поиск по отправителю или получателю.
«Тема» - поиск по теме письма.
«Содержимое» - поиск по содержимому. Например, можно ввести часть фразы из тела письма.
