Панель фильтрации

Панель фильтрации доступна во всех режимах отображения «Календаря». Активируется она в левой части экрана путем нажатия на специальную вкладку, как указано на рисунке:




Помимо выбора режимов отображения, панель дает возможность фильтровать задачи по всевозможным критериям.
Все поля активируются путем нажатия на них левой кнопкой мышки, а чтобы изменения вступили в силу, необходимо нажать на красную кнопку «Фильтровать», которая должна появиться чуть правее панели.




В панели предусмотрена фильтрация по имени ответственного сотрудника, а также по клиентам и авторам.




Поле «Название» соответствует колонке «Имя задачи».
Поле «Код» соответствует id задачи, который отображаетсяв виде колонки с заголовком «#».
Дата создания От и До позволяет задать временный промежуток для поиска нужной задачи.




Блок фильтрации для задач, которые включают в себя продукт, работает аналогично вышеописанному функционалу. Следует учитывать, что поля с продуктом не будут фильтровать задачи, в которых продукт отсутствует.




По умолчанию список задач включает отображение закрытых или выполненных задач. Это можно изменить, убрав галочку в checkbox`е «Показывать закрытые».
В этом же блоке находятся бизнес-процессы, по которым можно выполнять фильтрацию. Для этого нужно отметить галочкой определенный бизнес-процесс и нажать кнопку «Фильтровать».
Каждый бизнес-процесс состоит из этапов. Просмотреть их можно, нажав на бизнес-процесс левой кнопкой мышки. Поля этапов в выпадающем списке бизнес-процесса также представлены checkbox`ами, которые аналогично используются в качестве критериев для фильтрации.
Важно помнить: если свернуть поле бизнес-процесса, то галочки, поставленные в checkbox`ах этапов, сброшены не будут.
