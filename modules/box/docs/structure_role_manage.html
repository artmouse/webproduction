Управление ролями

Управление ролями является ключевым моментом в создании структуры компании. За данный функционал отвечает вкладка «Настройка ролей», которая находится в верхней части страницы:




Все созданные роли отображены в виде списка:




Древовидная структура сотрудников выстраивается благодаря колонке, которая называется «Подчиняется». Если в этом поле задано значение 0, это означает, что данная роль является корнем для дерева структуры.


Заголовки списка позволяют выполнять сортировки. Для этого нужно нажать на один из заголовков левой кнопкой мышки, после чего весь список будет отсортирован в алфавитном порядке. При повторном нажатии произойдет сортировка в обратном алфавитном порядке.
Полей у роли достаточно большое количество, поэтому все они вряд ли поместятся на экране. Для просмотра всех полей следует воспользоваться полосой прокрутки.
Также список можно экспортировать в форматы: CSV, XLS, XML. За это отвечает специальная опция внизу списка:




При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.

За создание новой роли отвечает форма «Создание новой записи». Выглядит она следующим образом:




Поле «Название роли» является обязательным. Если оставить его пустым — система не сможет создать новую роль и выдаст сообщение об ошибке.


Поле «Подчиняется» отвечает за выбор руководящей должности для данной должности. Если поле оставить пустым - это будет значить, что данная роль не подчиняется никому и является корневой для древовидной структуры.


Для того, чтобы применить изменения нужно нажать на кнопку «Вставить новую запись».


После этого в структуре появится новый блок с указанной должностью. Предположим, что новая роль называлась «Главный бухгалтер». В этом случае структура будет выглядеть так:




Блок «Главный бухгалтер» пока пуст. Чтобы добавить в него сотрудников нужно зайти в карточку контакта того сотрудника, который занимает данную должность.




Сначала нужно занести данный контакт в сотрудники, для этого нужно поставить галочку в checkbox`е «Сотрудник». Затем активировать поле «Должность» и ввести в нем занимаемую должность сотрудника. В нашем случае это будет «Главный бухгалтер».


Для внесения изменений нужно навести курсор мышки на поле, чтобы рядом с ним появился значок с карандашом. Нажмите на этот значок, внесите нужную информацию, затем нажмите на синий значок с галочкой, чтобы подтвердить изменения. Для отмены - выберите красный значок с крестиком.


Важно, чтобы указанная должность существовала в структуре ролей. В этом случае при переходе на страницу структуры имя сотрудника отобразится в нужном блоке.
