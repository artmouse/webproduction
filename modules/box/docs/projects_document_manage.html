Управление документами проекта

CRM система дает возможность формировать документы в контексте проекта. Карточка проекта предоставляет доступ к этому функционалу. В верхней части карточки, в горизонтальном меню, есть пункт «Документы», нажмите на него.




Далее на экране появится страница для управления документами.
Если на данный момент не было создано ни одного документа, то вместо списка документов будет отображена надпись «Документов пока нет. Нажмите добавить документ». Часть надписи подчеркнута пунктирной линией и является ссылкой, перейдя по которой можно попасть в форму создания нового документа.




Если в списке документов есть записи, то создать документ можно при помощи кнопки «Добавить документ»:




Затем откроется форма для создания документа, которая содержит следующие поля:

  • Название документа — обязательное поле. Система не сможет создать документ, если данное поле оставить пустым.
  • Шаблон документа — система включает несколько встроенных шаблонов на разных языках.
  • Юридическое лицо
  • Дата (Отправлен, Получен, Архив)
  • Загрузить файл оригинала — приложить файл с оригиналом документа.

Важно! Поле «Юридическое лицо» это отдельная настройка. При создании документа формирование реквизитов происходит именно благодаря ей.


После заполнения полей нужно нажать кнопку «Добавить».

Более подробно об отображении списком можно почитать в разделе «Календарь», либо по этой ссылке .

Списком можно управлять при помощи панели фильтрации. Активируется она в левой части экрана путем нажатия на специальную вкладку, как указано на рисунке:




Поля панели являются критериями фильтрации. К примеру, «Номер» — соответствует id документа, «Подгруппа» — отвечает за выбранный шаблон документа.
Панель также отображает номер проекта, к которому относятся документы.
Статусы документа — позволяют выполнять фильтрации, ориентируясь на текущий статус документа:

  • Сформирован
  • Отправлен
  • Получен
  • В архиве

В самом низу панели находятся поля, отвечающие за фильтрацию по дате.




Все поля панели активируются путем нажатия на них левой кнопкой мышки, а для того, чтобы изменения вступили в силу следует нажать на кнопку «Фильтровать».


Для того, чтобы отредактировать созданный ранее документ, необходимо нажать на номер документа или его название:




Далее появится форма для редактирования документа. Выглядит она так:




Форма условно разделена на два блока. Правый блок служит для просмотра и редактирования тела документа.




В левом блоке можно отредактировать основные поля документа:

  • Номер документа
  • Название документа
  • Тип документа
  • Дата создания документа
  • Дата отправки документа
  • Дата получения документа обратно
  • Дата архивации документа
  • Срок окончания действия документа
  • Авторы документа
  • Юридическое лицо

Важно! Поле «Юридическое лицо» это отдельная настройка. При создании документа формирование реквизитов происходит именно благодаря ей.


После внесения всех изменений следует нажать на кнопку «Сохранить».
