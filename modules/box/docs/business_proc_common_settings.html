Общие настройки бизнес-процесса

За создание и редактирование бизнес-процесса отвечает специальная форма «Редактирование бизнес-процесса».


Чтобы начать работу с формой нужно активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Настройки» → «Бизнес-процессы».




Следующая страница включается в себя форму для создания нового бизнес-процесса, а также список всех существующих бизнес-процессов.


Для внесения изменений в какой-либо бизнес-процесс нужно нажать на его название левой кнопкой мышки.

Если переход к форме произошел при создании нового бизнес-процесса, то можно заметить, что поля «Название процесса» и «Предназначен для» — дублируются. Данные поля являются обязательными.


Поле «Предназначен для» позволяет определить, для чего предназначен данный бизнес-процесс. Это позволит создать задачу, проект или заказ со структурой этапов, заданной в бизнес-процессе.


Поле «Список продуктов» является полем для поиска продуктов. Чтобы им воспользоваться, нужно ввести название продукта и подождать, пока система выдаст возможные варианты наименований из базы. Добавление продукта происходит по клику мышкой.


Бизнес-процесс можно установить по умолчанию для новых заказов или задач. В этом случае при создании заказов и задач не надо будет отвлекаться на указание бизнес-процесса.


Также есть возможность назначить ответственного для старта новых задач, созданных в контексте данного бизнес-процесса, и регулировать срок жизни бизнес-процесса.




За удаление бизнес-процесса отвечает вкладка «Удалить», которая находится в верхней части экрана. Вкладка становится доступной только при работе с формой «Редактирование бизнес-процесса».




После внесения всех изменений нужно нажать на кнопку «Сохранить».
