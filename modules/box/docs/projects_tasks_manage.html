Управление задачами проекта

В рамках одного проекта может быть назначено множество задач. Для управления задачами из проекта следует активировать выдвижную панель, которая находится в левой части экрана. Затем нажать левой кнопкой мышки на пункт «Проекты».




В появившемся списке нужно выбрать один из проектов. Для этого нужно кликнуть на его название или номер, как показано на рисунке:




После этого на экране появится форма для редактирования проекта. В верхней части формы, в горизонтальном меню, есть пункт «Задачи», нажмите на него.




Откроется страница с формой для быстрого создания задачи. Эта форма обладает минимальным набором полей, а именно:

  • имя задачи
  • ответственный за выполнение
  • выполнить до
  • бизнес-процесс
  • что нужно сделать?

Поля «Имя задачи» и «Бизнес-процесс» обязательны для заполнения. Если эти поля оставить пустыми, система НЕ выдаст сообщение об ошибке, но задача создана НЕ будет.
Следствием того, что задача создана корректно, является ее отображение в списке задач.
После того как все поля формы будут заполнены, нажмите на кнопку «Создать задачу».




Также на странице задач проекта, чуть ниже блока «Быстрое создание задачи», находится список всех задач.




Для выбора конкретной задачи следует нажать на ее название или номер левой кнопкой мышки.
Более подробно об отображении списком можно почитать в разделе «Календарь», либо по этой ссылке .

После нажатия на название задачи должна появиться страница редактирования задачи из проекта. Выглядит она следующим образом:




Вернуться обратно к проекту можно нажатием на вкладку с названием проекта вверху экрана или воспользовавшись пунктом «Проект», в горизонтальном меню.
Вкладка «Назад» вернет пользователя к предыдущей странице.
Блок «Структура» показывает, какие задачи относятся к данному проекту. Текущая задач выделена серым фоном, а для выбора другой задачи нужно просто нажать на ее название в списке.
Все остальное место на экране занимает форма для редактирования задачи.




Для внесения изменений в форму нужно навести курсор мышки на конкретное поле, чтобы рядом с ним появился значок с карандашом.




Нажмите на значок, внесите нужную информацию и нажмите на синий значок с галочкой, чтобы подтвердить изменения. Чтобы отменить изменения, выберите красный значок с крестиком.




В блоке для написания комментариев можно создавать подзадачи. Более подробно об этом функционале можно прочесть в разделе «Задачи» → «Создание подзадач» или по этой ссылке .
