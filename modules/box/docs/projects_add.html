Добавление проекта

Для добавления проекта следует активировать выдвижную панель, которая находится в левой части экрана. Затем нажать левой кнопкой мышки на пункт «Проекты».




В списке всех проектов следует воспользоваться закладкой «Создать проект».




После чего на экране появится форма добавления нового проекта.




Поля «Название проекта» и «Бизнес-процесс проекта» являются обязательными, то есть система не сможет создать новый проект, если не указать эти поля.


Все остальные поля являются необязательными для заполнения, однако, несут в себе уточняющий характер, который немаловажен для ответственных по проекту.




После того как все поля формы были заполнены, завершить процесс создания проекта нужно нажатием на кнопку «Сохранить» в нижней части экрана.


Для того, чтобы после создания проекта система выполнила перенаправление на создание нового проекта, следует воспользоваться кнопкой «Сохранить и добавить еще».
