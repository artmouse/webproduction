Просмотр структуры

Чтобы просмотреть структуру компании следует активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Структура»:




Страница структуры изначально отображена в виде дерева ролей сотрудников. Роль в рамках системы определяет права и должность сотрудника.


Чем выше должность, тем выше она находится в древовидной структуре. Равные должности выстраиваются в один уровень по горизонтали. Подчиняющиеся должности располагаются под руководящими.




Имена сотрудников отображаются в блоке должности, к которой они принадлежат. При наведении курсора на имя система показывает краткую информацию о сотруднике.




С блоком информации тоже можно взаимодействовать. Так, к примеру, при нажатии на Email сотрудника система выполнит переход к форме отправки письма, при нажатии на «Написать SMS» - к форме отправки SMS сотруднику.




Для того, чтобы посмотреть карточку контакта, нужно кликнуть по имени сотрудника. Система выполнит переход на страницу профиля, в котором можно посмотреть всю персональную информацию о сотруднике.
Более подробно о карточке контакта можно почитать в разделе «Контакты» или по этой ссылке .


Также структуру ролей можно отобразить списком. Для этого нужно нажать на кнопку «Список» в верхней части экрана:




После этого страница структуры будет выглядеть таким образом:




Данный способ отображения удобен тем, что позволяет выполнять сортировки по заголовкам контактов. Для этого нужно нажать на заголовок левой кнопкой мышки, после чего список будет отсортирован в алфавитном порядке. При повторном нажатии произойдет сортировка в обратном алфавитном порядке.




Поля с электронными почтами(E-mail) сотрудников, как и при отображении деревом, служат для перехода к форме отправки письма.




Чтобы посмотреть карточку контакта, нужно кликнуть по имени сотрудника. Система выполнит переход на страницу профиля, в котором можно посмотреть всю персональную информацию о сотруднике.
Более подробно о карточке контакта можно почитать в разделе «Контакты» или по этой ссылке .
