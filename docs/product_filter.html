Фильтры товаров

В горизонтальном меню карточки товара существует вкладка «Фильтры», которая позволяет определить, какие фильтры будут использоваться для данного товара. Для того, чтобы добавлять фильтры к товару их необходимо предварительно создать.


Чтобы перейти на вкладку нужно нажать на нее левой кнопкой мышки:




Страница, где фильтры подключается к товару выглядит так:




Одна строка формы соответствует одному фильтру.
В выпадающем списке можно выбрать название фильтра, в поле правее - его значение.
Checkbox «Доступен в фильтрах» - отвечает за отображении фильтра в панели фильтрации при поиске нужного продукта на сайте. Это позволяет пользователям выполнять сортировки списка продуктов по различным критериям, тем самым облегчая поиск нужного товара.
Checkbox «Характеристика в таблице» - отвечает за отображение фильтров-характеристик в карточке товара. Эти данные для удобства выводятся в отдельном блоке.


После того как все необходимые фильтры были добавлены, следует нажать на кнопку «Сохранить», которая находится в нижней части экрана. Система уведомит пользователя о том, что данные успешно сохранены.


Также с текущей страницы можно создавать фильтры. Для этого, под списком всех фильтров, находится кнопка «Добавить фильтр-характеристику». Более подробно о том, как создавать фильтры можно почитать в разделе «Просмотр фильтров для товаров и их создание» или по этой ссылке .
