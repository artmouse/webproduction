Панель фильтрации

Продуктами можно управлять при помощи панели фильтрации. Это удобно для быстрого поиска конкретного продукта среди большого количества наименований. Активируется панель в левой части экрана нажатием на специальную вкладку, как указано на рисунке:




После этого экран будет выглядеть так:




Все поля панели являются критериями для фильтрации и активируются путем нажатия на них левой кнопкой мышки. Для того, чтобы изменения вступили в силу следует нажать на кнопку «Фильтровать», которая появится чуть правей панели.




Если в результате фильтрации список продуктов не был сформирован — это значит, что продуктов соответствующих заданным критериям в базе не существует.


Панель фильтрации состоит из следующих полей:


Поля «Код», «Артикул», «Штрих-код», «Модель», «Наименование», «Теги» соответствуют аналогичным полям в списке продуктов. В поле нужно ввести имя продукта или его характеристику после чего система выполнит фильтрацию списка и выведет все возможные варианты.
При помощи checkbox`ов можно отфильтровать скрытые, удаленные или синхронизируемые товары.
Поле «Бренды» представлено выпадающим списком, как и поле «Поставщик».
В самом низу панели находится блок категорий. При нажатии на категорию список сразу же меняется, отображая только те товары, которые входят в данную категорию.
Поле «Введите название категории» служит для поиска нужной категории. Укажите в поле название категории и список категорий под полем будет отсортирован.
