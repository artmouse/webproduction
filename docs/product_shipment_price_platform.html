Выгрузка товара на прайс-площадки

CRM система OneBox позволяет генерировать прайсы для различных прайс-площадок. Настройка прайс-площадок происходит через меню «Настройки» → «Прайс-площадки». Более подробно о данном функционале можно почитать в разделе, посвященному прайс-площадкам или по этой ссылке.
В системе предусмотрена возможность добавлять товар в прайс прямо из карточки продукта. Для этого существует вкладка «Прайс-площадки».
Чтобы перейти на вкладку нужно нажать на нее левой кнопкой мышки.




Страница с прайс-площадками выглядит следующим образом:




Все площадки объединены в поле «Выгружать продукт на прайс-площадки» и представляют собой checkbox`ы.


Для каждой площадки прайс формируется из определенного перечня продуктов. Отметив один или несколько checkbox`ов, можно добавить текущий продукт в прайс, который будет формироваться для данной прайс-площадки.
Важно! Товар не будет добавлен мгновенно. Добавление товара в прайс произойдет через час - это связано с некоторыми особенностями в работе системы.


Для сохранения изменений обязательно воспользуйтесь кнопкой «Сохранить».
