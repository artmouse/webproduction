Страницы


Просмотр всех страниц

Функционал системы позволяет пользователю создавать собственные текстовые страницы, используя админ-панель. Такие страницы дают возможность формировать информационный блок: размещать текстовый контент, не требующий логики, а также контент с логикой. К последнему можно отнести: новостную ленту, галерею, отзывы, блок часто задаваемых вопросов и так далее. Все это может оказаться полезным в работе интернет-магазина.

Добавление страницы

В списке слева отображаются все созданные страницы. Для добавления новой страницы воспользуйтесь вкладкой в верхней части экрана:




Форма для добавления страницы выглядит следующим образом:




«Название страницы» - обязательное поле, нужно для указания названия страницы в админ-панели.
«Содержимое страницы» - редактор для работы с текстовым контентом. Позволяет форматировать и стилизовать текст, также проверять правильность написания подобно тому, как это реализовано в популярных текстовых процессорах.
«Название кнопки в меню» - обязательное поле, отвечает за название страницы в меню (находится в "шапке" или в "подвале" сайта).
«Родительская страница» - задать иерархию страниц для админ-панели.




«Тип (логика) страницы» - позволяет задавать структуру и логику для страниц. Например, «Вопрос-ответ (FAQ)» отображает страницу с вопросами и ответами на них, «Новости и статьи» служат для формирования новостной ленты. Страницы с логикой требуют дополнительной настройки. Такие настройки, как правило, находятся в меню OneBox → «Настройки». Например, настройки новостной ленты объединены в пункте «Новости, статьи, блог». Более подробно об этом функционале можно прочесть по этой ссылке . Если не указать логику, контент будет отображаться в виде обычной страницы.
«Скрытая» - скрыть страницу.
«Порядок сортировки» - числовое значение. 0 — первый пункт в списке, чем выше число, тем ниже позиция пункта в списке всех страниц.
Следующий блок полей отвечает за SEO-оптимизацию контента. Заполнение этих полей предусмотрено для поднятия позиции сайта в результатах поисковых систем и требует базовых знаний в области SEO.


Для сохранения изменений нажмите на кнопку «Вставить новую запись», которая находится в нижней части экрана.


Результат добавления новой страницы можно проверить на главной странице магазина.




Редактирование и удаление страниц

При выборе конкретной страницы справа от списка будет отображаться форма для редактирования страницы.




Форма для редактирования обладает аналогичным набором полей, что и форма для добавления. Также данная форма позволяет удалить страницу. Для этого нужно воспользоваться кнопкой «Удалить запись», которая находится в нижней части экрана.




Далее система запросит подтверждение об удалении в виде модального окна.
