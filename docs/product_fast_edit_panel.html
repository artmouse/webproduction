Панель быстрого редактирования продуктов

При просмотре всех продуктов существует возможность вносить изменения, используя панель быстрого редактирования.


По умолчанию панель быстрого редактирования скрыта. Для ее активации нужно отметить нужный продукт галочкой. Скриншоты демонстрируют, как это можно сделать в зависимости от выбранного режима отображения.

Панель быстрого редактирования позволяет выполнять следующие действия:


  • «Переместить в категорию» — переместить выделенный продукт или группу продуктов в категорию.
  • «Переместить в бренд» — группировка товара по бренду, позволяет быстро добавить товар в бренд без лишних переходов на вкладку брендов.
  • «Синхронизируемый/не синхронизируемый» — синхронизация с 1С.
  • «Открыть/Скрыть» — сделать товар невидимым.

Также есть поля для удаления/добавления тегов, удаления/восстановления самого товара, для добавления картинки, вставки описания и указания наличия товара.

  • Кнопка «Создать новый заказ» позволяет добавить продукт в новый заказ. Если галочка «Перейти в заказ» установлена, то система выполнит переход к форме редактирования заказа.
  • Кнопка «Добавить в существующий заказ» позволяет добавить отмеченные товары в конкретный заказ, но для этого нужно ввести его номер в поле «id».

Для подтверждения всех изменений кликните на кнопку «Изменить».
