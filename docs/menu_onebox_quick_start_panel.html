Панель быстрого старта

Пункт «Создать...» в меню OneBox также называют панелью быстрого старта. Она дает доступ к созданию контента и всевозможных процессов, к написанию электронных писем и сообщений в техподдержку.


Для того, чтобы активировать панель, нажмите на пункт «Создать...» левой кнопкой мышки:




Панель включает в себя множество подпунктов, каждый из которых охватывает определенный функционал системы.


Добавить контакт — переход к форме добавления контакта (подразумевается добавление персоны). Более подробно о контактах можно почитать в соответствующем разделе документации или по этой ссылке .


Добавить компанию - переход к форме добавления контакта, в которой установлено поле «Тип» - компания. О полях формы можно почитать по этой ссылке .


Добавить проект — переход к форме добавления проекта, которая находится в «Проекты» → «Создать проект». Более подробно о создании проектов можно почитать в разделе «Проекты» → «Добавление проекта» или по этой ссылке .


Добавить заказ - переход к форме добавления заказа, которая находится в «Заказы» → «Добавить новый заказ». Более подробно о создании заказов можно почитать в разделе «Заказы» → «Добавление нового заказа» или по этой ссылке .


Добавить платеж — переход к форме добавления платежей, которая находится в разделе «Финансы». Более подробно о создании платежей можно почитать в разделе «Финансы» → «Добавление платежа» или по этой ссылке .


Добавить продукт - переход к форме добавления продукта, которая находится в «Продукты» → «Новый продукт». Более подробно о создании продуктов можно почитать в разделе «Продукты» → «Добавление продукта» или по этой ссылке .


Написать в техподдержку — переход к форме составления письма для техподдержки. Как этим пользоваться, можно почитать в разделе «OneBox меню» → «Написать в техподдержку» или по этой ссылке .


Написать письмо — переход к форме для отправки электронных писем, которая выглядит таким образом:




Для того, чтобы вернуться на предыдущую страницу, воспользуйтесь кнопкой «Х», которая находится в левой верхней части экрана.


Поле «Кому» - является обязательным для заполнения. Если его не заполнить, система не сможет выполнить отправку письма и выдаст ошибку.
«От кого» - поля для обратной связи.
«Тема письма» - заголовок письма.
«Сообщение» - поле для ввода тела сообщения.
«Добавить файл» - если есть необходимость можно прикрепить к письму файлы.


После заполнения всех полей нажмите на кнопку «Отправить». Кнопка «Отмена» - служит для возвращения на предыдущую страницу и работает аналогично кнопке «Х», которая была описана выше.


Добавить задачу - переход к форме добавления задачи, которая находится в «Задачи» → «Создать задачу». Более подробно о создании задачи можно почитать в разделе «Задачи» → «Добавление задачи» или по этой ссылке .
