Импорт и экспорт Excel

CRM система OneBox позволяет импортировать и экспортировать товары в формат XLS. Чтобы воспользоваться данным функционалом необходимо активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Продукты».




Далее кликните на пункт «Импорт и экспорт Excel».




Также на страницу импорта и экспорта товаров можно попасть из списка всех товаров. Для этого в списке всех товаров воспользуйтесь вкладкой «Импорт и экспорт Excel», которая находится в верхней части экрана:




Система выполнит переход к странице импорта и экспорта контактов, которая выглядит следующим образом:




Страница включает пошаговую инструкцию о том, как правильно пользоваться функционалом.
Выпадающий список позволяет выбрать, с какой группой продуктов будет работать система. В поле «Send XLS files to emails» нужно указать email, на который придет сформированный файл XLS.
Для подтверждения изменений нажмите на кнопку «Скачать все данные в Excel».




Процесс загрузки контактов из XLS также интуитивно понятен.
Прикрепите файл XLS при помощи кнопки «Обзор...», затем нажмите на «Закачать все данные из Excel». После этого система добавит контакты в базу системы.
