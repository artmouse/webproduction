Списки

Для удобства управления продуктами CRM система OneBox позволяет объединять их в списки. Это удобно для тематического разделения продуктов, с целью привлечения к ним внимания. Так, к примеру, товары-подарки, которые продавались наиболее успешно, могут быть объединены в список «Лучшие подарки 2015».
Списки могут отображаться на главной странице сайта. Такое поведение задается в настройках.


В карточке товара можно проследить занесен ли данный товар в какой-либо из списков или нет. За это отвечает вкладка «Списки»:

Если товар не относится ни к одному из списков, то таблица будет пуста. Важно! Также нужно помнить, что товар может быть занесен в несколько списков одновременно.


Все поля таблицы можно править только при помощи формы для редактирования списка. Для того, чтобы перейти к ней нужно нажать на название списка в колонке «Название списка с продуктами». Название является ссылкой. После этого система выполнит переход к форме для редактирования списка.
Единственное поле, которое можно менять прямо в таблице это «Способ отображения». Для этого нужно кликнуть на ячейку левой кнопкой мышки.

Чтобы добавить новый список нужно воспользоваться специальным функционалом для создания списков. За такой функционал отвечает вкладка «Списки продуктов», ее можно найти при просмотре всех продуктов. Более подробно про объединение продуктов в списки можно почитать в разделе «Списки продуктов» или по этой ссылке .
