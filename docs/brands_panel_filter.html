Панель фильтрации брендов

Панель фильтрации в списке брендов может пригодиться для быстрого поиска нужного бренда или для сортировки по определенному критерию. По умолчанию она скрыта. Чтобы ее активировать нужно нажать на кнопку, которая находится слева от списка брендов, см. рисунок ниже:




После активации панель будет выглядеть так:




Теперь можно фильтровать данные списка. Для этого в одном из полей, к примеру, «Имя бренда» вводим нужный продукт, после чего жмем красную кнопку «Фильтровать» или клавишу «enter»:




Если данные введены корректно, то список брендов изменится и покажет желаемый результат, в случае, если в данных была допущена ошибка или таких данных не существует, система отобразит пустой список.
