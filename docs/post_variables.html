Переменные в письмах

При сипользовании действий - отправки Email-писем, в шаблонах и темах писем можно использовать переменные.



Доступные переменные:
[company] - название компании клиента
[name] - ФИО клиента
[name_smart] - "умное" имя
[name_first] - имя клиента
[name_last] - фамилия клиента
[name_middle] - отчество клиента
[name_first_last] - имя и фамилия клиента
[contractordetails] - реквизиты для оплаты
[orderid] - номер заказа
[ordercurrency] - валюта заказа
[ordername] - название заказа
[ordersum] - сумма заказа
[orderdateto] - дата выполнения заказа
[ordercdate] - дата создания заказа
[file-ID] - подстановка прямого URL на файл с заданным ID
