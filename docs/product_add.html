Добавление продукта

Добавить новый продукт можно при помощи выдвижной панели, которая находится в левой части экрана. Наведите на нее курсор, затем нажмите левой кнопкой мышки на пункт «Продукты».




Для добавления продукта кликните на пункт «Новый продукт».




Также добавить новый продукт можно в списке всех продуктов. Для этого служит вкладка «Новый продукт», которая находится в верхней части экрана:




После этого система выполнит переход к форме редактирования продукта.


Среди всех полей формы единственным обязательным для заполнения является поле «Наименование». Если это поле оставить пустым - система не сможет создать новый продукт и выдаст сообщение об ошибке.


Все поля разделены на блоки, которые можно скрывать. Чтобы скрыть какой-то блок нужно нажать на него левой кнопкой мышки и после короткой анимации тело блока исчезнет. Видимым остается только заголовок, кликнув по которому можно снова развернуть содержимое блока.




Слева от общей информации находятся поля, отвечающие за изображения товара. Важно! Размер загружаемых изображений не должен превышать больше 10 Мбайт.




В общей информации задаются основные характеристики товара. Категория и бренд — представлены в виде выпадающих списков. Еще здесь можно установить единицы измерения для товара, его цену, закупочную цену.


Два поля служат для полного и краткого описания. Причем поле для полного описания представляет собой редактор, в котором можно выполнить базовые операции по форматированию текста.




Блок «Наличие, склад» - отвечает за наличие товара, а также с его помощью можно задавать время жизни товара на складе.
«Акции и подарки» - скидка на товар в процентах, также позволяет выводить перечеркнутую цену.




«Опции заказа» - позволяет каждому клиенту задать свой уровень цен.




«Упаковка продукта» - блок, в котором можно указать модель продукта, его серию и габариты.
«Скачиваемый продукт» - контент, который становится доступным к скачиванию только после того, как продукт был куплен клиентом.




После заполнения всех полей нужно нажать на кнопку «Создать продукт». Система сохранит все изменения в базе и откроет форму заново уже с заполненными полями. Это сделано специально, чтобы пользователь мог перепроверить все поля и удостовериться, что все данные указаны корректно.


Для того, чтобы перейти к списку всех продуктов нажмите на кнопку «К списку продуктов».
