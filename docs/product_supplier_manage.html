Управление поставщиками

Управление поставщиками товара происходит через вкладку «Поставщики». Вкладка находится в горизонтальном меню карточки товара. Перейти на нее можно, кликнув левой кнопкой мышки.

Поставщики отображаются в виде таблицы. Одна строка таблицы отображает одного поставщика. При наведении курсора мышки - строки таблицы подсвечиваются для наглядности.


Большинство полей активируются нажатием левой кнопкой мышки. Поле с названием поставщика является выпадающим списком, который дает возможность добавлять или удалять поставщика для текущего продукта.
Список поставщиков формируется в пункте «Настройки» → «Поставщики». Более подробно о данном функционале можно почитать в разделе, связанным с поставщиками или по этой ссылке.
Важно! Некоторые поля таблицы, к примеру, «Время доставки» можно редактировать только из настроек поставщика.


Также в таблице можно указывать цену на товар, скидку, которую делает поставщик и наличие товара.


РРЦ (рекомендуемая розничная цена) - если она указана у поставщика, по которому идет пересчет цен (по наценкам), то вне зависимости от результата наценки ставится данная цена.


МРЦ (минимальная розничная цена) - если при наценке по данному поставщику цена с наценкой выходит меньше МРЦ, то товару ставится цена МРЦ.
