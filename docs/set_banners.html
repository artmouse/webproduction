Баннера


Таблица баннеров

В системе предусмотрена возможность управления баннерами. На страницах ресурса зарезервированы специальные области, куда будет помещено рекламное изображение. К примеру, такая область есть на главной странице, также на страницах с категориями, брендами и не только. Для установки баннера нужно иметь правильно отформатированное графическое изображение. В случае, если для баннера нужно задать другое расположение, отличное от того, что задано по умолчанию, следует обратиться в техподдержку. О том, как это сделать можно прочесть в разделе «Настройки» → «Написать в техподдержку» или по этой ссылке .


Управление баннерами происходит через настройки системы. Активируйте меню OneBox, которое находится в левой части экрана, выберите пункт «Настройки», затем в выпадающем списке найдите пункт «Баннера» и нажмите на него левой кнопкой мышки:




Следующая страница выглядит таким образом:




Все записи объединены в таблицу, которая позволяет сортировать данные. Происходит сортировка по нажатию на заголовок колонки. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок.
Под таблицей находится функционал для экспорта документа в форматы: CSV, XLS, XML. Это реализовано с помощью меню, которое активируется по клику мышкой.


При наведении курсора на заголовок таблицы изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяет регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.




Добавление баннера

Для добавления баннера воспользуйтесь вкладкой «Добавить баннер», она находится в верхней части экрана:




Далее появится форма для добавления:




«Название» - обязательное поле при создании баннера.
«Скрытый баннер» - скрыть баннер со страницы.
«Сортировка» - числовое значение для установки приоритета баннеров.
«Картинка» - загрузка изображения баннера. Важно! Размер картинки должен быть в точности таким, как указано в поле «Расположение», иначе изображение подвергнется ресемплированию вследствие чего будет растянуто и потеряет презентабельный вид.
«Ссылка URL» - обязательное поле с ссылкой, куда произойдет переход при нажатии на баннер.
«Расположение» - выпадающий список с различными зарезервированными позициями, в которых будет отображаться баннер. Названия позиций говорят сами за себя, например:

  • Сверху в категории и бренде — при просмотре категории или бренда баннер будет отображаться в верхней части страницы.
  • На всю ширину под главным меню — отображения баннера на главной странице.
  • Слева в колонке — отображение баннера под панелью фильтров.
  • Добро пожаловать - баннер появится в виде отдельного окна при первом посещении сайта.

«Категории» - указать категорию, в которой будет отображаться баннер.
«Комментарий» - подпись для баннера.




«Начало показа», «Окончание показа» - задать время, когда будет отображаться баннер.
Для сохранения изменений нажмите на кнопку «Вставить новую запись», которая находится в нижней части экрана.





Редактирование и удаление баннера

Быстрые правки можно вносить прямо в таблице, в которой отображаются все баннера. Для этого нужно нажать левой кнопки мышки на ячейку, после чего она станет активной:




Затем, чтобы изменения вступили в силу нажмите клавишу «Enter».


Для перехода к форме редактирования кликните на название баннера в таблице.




Форма для редактирования обладает аналогичным набором полей, что и форма для добавления. После внесения любых поправок нужно нажать на кнопку «Сохранить запись». Также данная форма позволяет удалить конкретный баннер. Для этого нужно воспользоваться кнопкой «Удалить запись», которая находится в нижней части экрана.





Статистика кликов

При редактировании баннера есть возможность просмотреть статистику кликов по нему. Для этого нужно нажать на вкладку «Статистику кликов», она находится в верхней части экрана.




Статистика отображается в виде графика и таблицы:




График показывает динамику кликов за определенный промежуток времени. Их количество можно увидеть при наведении указателя мышки на конкретный столбец.




Таблица отображает более детальную информацию о том, с какого IP-адреса заходил пользователь, его логин, сессия и точное время. Колонка с логином является ссылкой и позволяет переходить к карточке контакта по клику.
