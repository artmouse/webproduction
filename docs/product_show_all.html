Просмотр всех продуктов

Чтобы попасть на страницу всех продуктов, необходимо активировать выдвижную панель, которая находится в левой части экрана, затем нажать левой кнопкой мышки на пункт «Продукты».




Пункт «Продукты» включает в себя множество подпунктов. Каждый подпункт охватывает определенный функционал, связанный с продуктами. На страницу всех продуктов можно попасть двумя способами:

  • нажать на «Продукты» - отображение продуктов и их категорий плиткой
  • нажать на «Продукты списком» - отображение списком, в котором присутствуют только продукты без учета категорий

Отображение продуктов и их категорий плиткой

При отображении продуктов плиткой экран будет выглядеть так:




Категории отображаются в виде папок, все остальные наименования — товары. Если картинка у товара отсутствует, то на ее место будет вставлена заглушка в виде изображения фотоаппарата.
Категории всегда отображаются первыми. Затем обязательно присутствует кнопка для создания новой категории, которая выглядит как папка со знаком «+».




Это удобно для быстрого создания категории. Нажав на кнопку, на экране появится модальное окно, в поле которого нужно указать имя для новой категории:




Нажмите для подтверждения «ОК» и после этого система создаст новую категорию.


Кнопка «Добавить категорию» присутствует в любой категории. Поэтому включить в категорию несколько вложенных подкатегорий не составит труда. Для этого кликните на нужную категорию, затем аналогично воспользуйтесь кнопкой «Добавить категорию», как было описано выше.


Перечень всех товаров, по аналогии с категориями, замыкает кнопка для добавления товаров.




При нажатии на нее система выполнит переход к форме добавления товара. Более подробно о данной форме и о том, как происходит процесс добавления товаров можно почитать по этой ссылке .


Плитка не разделена на страницы при помощи пагинатора, она тянется вниз пока не будут отображены все товары и категории, которые существуют в базе данных. Перемещение при данном способе отображения возможно при помощи полосы прокрутки браузера.




Также на данной странице можно управлять товарами и категориями при помощи метода drag-and-drop. Для этого нужно навести курсор на товар или категорию и зажать левую кнопку мышки. Далее следует двигать курсор в нужном направлении и следить за желтым фоном, который определяет конечную точку перемещения. Цвет категорий при этом изменится.




При помощи данного функционала можно быстро вкладывать товары в категории или создавать иерархию категорий.


Чтобы вытащить таким образом товар из категории нужно переместить его на кнопку «Назад». Когда фон кнопки изменится на желтый, можно отпустить левую кнопку мышки. После этого товар будет перемещен на один уровень вверх.

Отображение продуктов списком

При нажатии на пункт «Продукты списком» система выполнит переход к списку всех продуктов. В данном случае речь идет о стандартном списке в системе OneBox.


Способ отображения списком позволяет сортировать продукты по различным полям, просматривать общее количество продуктов, также пользоваться панелью фильтрации и панелью быстрого редактирования.


Сортировка выполняется в алфавитном порядке или по убыванию/возрастанию порядкового номера продукта. Происходит она по нажатию на заголовок колонки в верхней части страницы. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок:




При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.


В самом низу списка можно увидеть общее количество продуктов.
Также здесь находится пагинатор, который позволяет листать продукты постранично. Для этого следует выбрать номер страницы или воспользоваться кнопками «Назад» и «Вперед» для перехода на предыдущую/следующую страницу соответственно.
