Уведомления

Пункт «Уведомления» в меню OneBox отвечает за отображение уведомлений. Данный функционал нужен для отслеживания изменений в процессах, в которых пользователь принимает непосредственное участие. Примером такого уведомления может быть комментарий, оставленный кем-то в задаче.

Для отображения уведомлений не нужно обновлять страницу, система самостоятельно в фоновом режиме отслеживает наличие изменений, касающихся пользователя.


Для просмотра уведомлений нужно нажать на пункт «Уведомления» левой кнопкой мышки. После этого рядом с меню OneBox появится дополнительная панель, в которой можно увидеть задачи, проекты или заказы с изменениями.
Если, к примеру, в одной задаче произошло несколько изменений, то все эти изменения будут объединены в один блок уведомлений.
Также все уведомления сортируются по группам: комментарии, электронные письма, звонки, уведомления и изменения. Группа отображается над блоком уведомлений:

Чтобы закрыть уведомление, нажмите на красную кнопку «Х». Она появится только в том случае, если курсор мышки наведен на уведомление.
