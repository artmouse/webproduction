Новости, статьи, блог


Просмотр всех новостей

Новостная лента часто оказывается неотъемлемой составляющей любого веб-ресурса. Возможность публиковать статьи, новости, также ведение блога по умолчанию включено в систему. Прежде, чем начать работу с новостной лентой стоит ознакомиться с функционалом страниц, который существует в системе. Подробней о нем можно прочесть в разделе «Настройки» → «Страницы» или по этой ссылке .

После создания страницы для новостной ленты можно перейти к созданию непосредственно самих новостей. Для этого активируйте меню OneBox, которое находится в левой части экрана, выберите пункт «Настройки», затем в выпадающем списке найдите пункт «Новости, статьи, блог» и нажмите на него левой кнопкой мышки:




Далее на странице можно увидеть таблицу со всеми новостями, которые уже были добавлены.




Таблица позволяет сортировать данные. Происходит сортировка по нажатию на заголовок колонки. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок.
Под таблицей находится функционал для экспорта документа в форматы: CSV, XLS, XML. Это реализовано с помощью меню, которое активируется по клику мышкой.


При наведении курсора на заголовок таблицы изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяет регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.




Добавление новости

Для добавления новости воспользуйтесь вкладкой «Добавить новость», которая находится в верхней части экрана:




Далее на экране появится форма для добавления:




«Название (заголовок)» - поле обязательное для заполнения, без него система не сможет создать новую статью/новость.
«Короткое содержимое» - вступительный текст, который будет отображаться в новостной ленте.
«Содержимое» - основной текст новости.
«Картинка» - используется для загрузки картинок.
Checkbox «Скрытая» - позволяет скрыть новость из новостной ленты.
«Создана» - дата публикации новости/статьи.
«Статья о продукте» - начните вводить название продукта и система выдаст список всех наименований с похожими названиями. При выборе продукта он будет отображаться сразу под новостью.
Также можно указать категорию и бренд, к которым относится новость.


Следующий блок полей отвечает за SEO-оптимизацию контента. Заполнение этих полей предусмотрено для поднятия позиции сайта в результатах поисковых систем и требует базовых знаний в области SEO.




Для сохранения изменений нажмите на кнопку «Добавить новость», которая находится в нижней части экрана.

Редактирование и удаление новости

Для редактирования статьи/новости нужно кликнуть на ее название или номер в таблице всех новостей.




Форма для редактирования обладает аналогичным набором полей, что и форма для добавления. После внесения любых поправок нужно нажать на кнопку «Сохранить новость/статью». Также данная форма позволяет удалить конкретную новость. Для этого нужно воспользоваться кнопкой «Удалить новость/статью», которая находится в нижней части экрана.
