Акционные наборы

При работе с продуктами предусмотрена возможность объединять их в акционные наборы. Это нужно для группировки товаров с целью установить скидку для конкретного набора.
Данные о наборе отображаются в карточке товара в блоке «Рекомендуем купить». Таким образом покупатель сразу может оценить преимущества покупки такого комплекта и решить стоит ли его покупать.

За данный функционал отвечает вкладка «Акционные наборы», которая находится в горизонтальном меню товара:




Страница наборов выглядит так:




Если товар уже включен в набор, то на экране появится список всех наборов, в которые входит продукт.
В противном случае страница будет пуста.
Таблица комплектов включает в себя поле с названием комплекта, его номером, а также поле, которое делает комплект видимым или скрытым.
Название комплекта является ссылкой, нажав на которую можно перейти к списку продуктов, входящих в данный комплект.


Для добавления набора воспользуйтесь вкладкой «Добавить набор».




Форма для добавления набора выглядит так:




Форма состоит всего из двух полей: «Название» нового комплекта и checkbox «Скрытый набор». Название набора нужно указывать обязательно иначе система не сможет создать его.
Далее необходимо нажать на кнопку «Вставить новую запись».

После этого появится страница для добавления товаров в комплект.




Товар, в контексте которого создавался набор по умолчанию попадает в данный комплект.


Для того, чтобы найти и добавить продукт в комплект нужно ввести его название в поле ввода рядом с кнопкой «Выбрать/найти/создать продукт»:




В выпадающем списке будут отображены все возможные варианты продуктов, которые на данный момент существуют в базе. Если выпадающий список не появился, значит такого продукта не существует.
Для выбора продукта нужно кликнуть по его названию. Название продукта отобразиться в поле поиска. Одновременно в комплект можно добавить несколько продуктов.
Чтобы продукты попали в комплект нужно нажать на кнопку «Сохранить». После этого система обновит страницу и в комплекте появятся новые наименования.
Если какой-то из продуктов оказался лишним, но еще не был добавлен в комплект, то его можно удалить из поля, нажав на кнопку «х», как указано на рисунке:




Также удалить все отобранные, но недобавленные в комплект продукты можно при помощи кнопки «Очистить».




Для просмотра списка всех продуктов, которые можно добавить в комплект, нужно нажать на кнопку «Выбрать/найти/создать продукт...».




Здесь отображается информация о номере, названии, наличии и стоимости продукта.
В отличии от быстрого поиска данная страница позволяет фильтровать поиск по категории. Это полезно в том случае, если известно неточное название продукта или неизвестно вообще.
В остальном поиск происходит аналогично. В поле «Найти продукт» нужно указать точное или приблизительное имя продукта, после чего нажать на кнопку «Найти»:




Для выбора продукта нужно нажать на него левой кнопкой мышки. После этого система автоматически выполнит переход на предыдущую страницу. Теперь для добавления продукта в список нужно нажать на кнопку «Сохранить», как уже было описано выше.


На странице просмотра всех продуктов также есть возможность создавать новые продукты, чтобы затем добавить их в комплект. Для этого вместо кнопки «Найти продукт» нужно выбрать кнопку "Создать продукт". После чего на экране появится форма создания нового продукта.




«Название продукта» является единственным обязательным полем для заполнения. Если продукт не имеет названия - система не сможет его создать, а на экране появится модальное окно с надписью name.
Все остальные поля являются уточняющими.
После заполнения всех полей нажмите на кнопку «Создать».


Далее система автоматически выполнит переход на страницу быстрого поиска продукта. Теперь для добавления продукта в комплект нужно нажать на кнопку «Сохранить», как уже было описано выше.


Для удаления товаров из комплекта нужно отметить checkbox «Удалить из набора» напротив нужного товара, после чего нажать на кнопку «Сохранить».
