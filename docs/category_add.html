Добавление категории

Чтобы добавить новую категорию нужно при помощи всплывающего меню OneBox, которое находится в левой части экрана, выбрать «Продукты» → «Категории», затем вверху переключиться на вкладку «Добавить категорию», как на рисунке ниже:




После этого откроется страница «Создание новой записи», в которой присутствуют все поля для создания новой категории. Выглядит она так:




Из основных полей для создания категории можно отметить следующие:




Имя категории — единственное обязательное поле для создания категории, без него система не сможет создать новую запись и выдаст ошибку.
Родительская категория — выпадающий список, благодаря которому мы можем создать дочернюю категорию в уже существующей категории, создавая таким образом определенную иерархию.
Скрытая — позволяет при необходимости скрыть категорию.
Порядок сортировки — числовой параметр, который позволяет задать свой порядок для сортировки.




Описание категории — служит для короткого пояснения к категории.
Изображение — позволяет загрузить картинку для категории.
Кроп изображение — функция для быстрого обрезания картинок.
После того, как основные поля были заполнены, создать новую категорию нужно, нажав на кнопку «Вставить новую запись», см. рисунок ниже:




Далее система напишет, что запись успешно добавлена и автоматически выдаст страницу для редактирования записи, чтобы можно было еще раз перепроверить все заполненные поля. Если все данные были внесены корректно и не требуют каких-либо изменений, можно вернуться в менеджер категорий или список категорий, нажав на соответствующую вкладку вверху экрана.
