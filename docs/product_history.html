История просмотров и редактирования продукта

История просмотров

В горизонтальном меню карточки товара существует вкладка «Просмотры». Данная вкладка отображает динамику просмотров товара. Для перехода к функционалу кликните по закладке левой кнопкой мышки.




Страница «Просмотры» выглядит следующим образом:




Частота просмотров товара отображена в виде графика. Шкала графика меняется автоматически в зависимости от временного промежутка, указанного в панели фильтрации слева.


Если промежуток не превышает одного месяца, то на шкале «X» будут отмечены дни месяца, если больше — график будет отображать динамику просмотров по месяцам.


Два поля в панели фильтрации позволяют вручную задать временный диапазон для графика. Поля со временем активируются нажатием левой кнопки мышки, после чего появится календарь, в котором можно выбрать нужную дату. Значения дат можно также вписать при помощи клавиатуры.


Под графиком можно увидеть список просмотров с детальной информацией о каждом из них, а именно: точное время, код сессии, ip-адрес, с которого просматривали контент, также логин зарегистрированного пользователя.




История редактирования продукта

Вкладка «История» в горизонтальном меню отвечает за отображение истории редактирования продукта. Все изменения, которые были внесены в карточке продукта через форму редактирования будут отображены здесь.




Страница «История» выглядит так:




Таблица изменений позволяет отследить кто, когда и какие действия совершал по отношению к данному товару.
Записи в колонке «Пользователь» представлены ссылками, нажав на которые можно сразу же перейти к просмотру карточки контакта.
