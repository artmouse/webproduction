Создание копии продукта

В системе предусмотрен функционал для копирования продуктов. Добраться до него можно при помощи выдвижной панели, которая находится в левой части экрана. Наведите на нее курсор, затем нажмите левой кнопкой мышки на пункт «Продукты».




Для копирования продукта кликните на пункт «Создать копию продукта».




Также скопировать продукт можно в списке всех продуктов. Для этого служит вкладка «Создать копию продукта», которая находится в верхней части экрана:




Форма для копирования продукта выглядит следующим образом:




Для того, чтобы выполнить копирование нужно знать номер продукта, который нужно скопировать. Код или id нужно ввести в поле «Введите код продукта (ID)». После того, как код введен, нажмите на кнопку «Отправить».


В случае, если был введен код, которого не существует в базе — система выдаст сообщение об ошибке.
Если код корректен, то на экране появится форма для редактирования продукта. Более подробно о полях формы можно почитать в разделе «Добавление продукта» или по этой ссылке .


Все поля формы являются редактируемыми и активируются нажатием левой кнопки мышки. Копия товара обладает идентичными полями, что и оригинал. В поля необязательно вносить изменения, однако важно помнить, что номер или id такого товара будет другой, отличный от оригинала. Это сделано для того, чтобы товары можно было различать на уровне базы данных.


После внесения изменений в копию товара нажмите на кнопку «Создать копию продукта».
Чтобы вернуться к списку всех продуктов воспользуйтесь соответствующей кнопкой в нижней части экрана.
