Просмотр списка категорий

Еще одним альтернативным способом управления категориями является список категорий. В отличие от менеджера категорий список категорий не обладает компактным отображением данных, однако имеет свои плюсы. Так, к примеру, он позволяет сортировать, фильтровать и просматривать данные, не заходя в панель редактирования, а также экспортировать данные в форматы: XML, XLS и CSV.
Зайти в список категорий можно при помощи всплывающего меню OneBox, которое находится в левой части экрана. В нем выбираем «Продукты» → «Категории», затем вверху нужно переключиться на вкладку «Категории», как указано на рисунке ниже:




Из скриншота видно, что список категорий отображает категории по порядку: сначала родительскую категорию, с значением 0 в колонке «Родительская категория», затем все ее дочерние категории.
Также в списке можно просмотреть все поля, которые были заполнены при редактировании записи. Все колонки при этом могут не влазить в экран, поэтому стоит воспользоваться полосой прокрутки:




Название колонок в списке можно использовать для выполнения сортировок. Чтобы сделать сортировку нужно нажать на конкретное поле. К примеру, если нажать на колонку «Категория» список будет отображаться в алфавитом порядке, а при повторном нажатии - в обратном алфавитном порядке:




Для редактирования категории следует нажать на ее название, после чего откроется страница для редактирования записи.
Для того, чтобы воспользоваться функцией экспорта в форматы: XML, XLS и CSV, нужно пролистать страницу в самый низ, это можно сделать либо колесом мышки, либо полосой прокрутки браузера. Внизу можно увидеть нужную опцию, как на рисунке ниже:




Здесь же находится пагинатор, который позволяет листать категории постранично. Для этого следует выбрать номер страницы или воспользоваться кнопками «Назад» и «Вперед» для перехода на предыдущую/следующую страницу соответственно:




При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.
