Связанные продукты

В горизонтальном меню карточки товара существует вкладка «Связанные продукты». Этот функционал нужен для того, чтобы в карточке товара можно было вывести список связанных товаров в блоке «Рекомендуем», который имеет вид карусели.


Чтобы перейти на вкладку нужно нажать на нее левой кнопкой мышки.




Страница со связанными продуктами выглядит следующим образом:




Если у текущего продукта уже имеются связанные продукты, то такие продукты будут отображаться в виде списка.
Список состоит из большого количества колонок, поэтому он вряд ли поместится на экране. Для того, чтобы просмотреть все поля таблицы нужно воспользоваться горизонтальной полосой прокрутки:




Имя связанного продукта является ссылкой. При нажатии на него система выполнит переход к карточке продукта.
Для того, чтобы удалить связанный продукт нужно отметить его в поле checkbox, как указано на рисунке:




После этого под таблицей появится поле «Удалить отмеченные продукты из списка», которое также является checkbox`ом. Отметив данный checkbox и нажав на кнопку «Сохранить», система приступит к удалению связанных продуктов.
Важно! Система удаляет связи между продуктами, но не удаляет продукты из базы данных.

Для создания связей между продуктами существует поле «Добавить еще продукты к списку». Поле предназначено для вставки номера или id продукта, с которым требуется установить связь. Посмотреть номер продукта можно в списке всех продуктов, за это отвечает колонка «Код».
Важно указывать корректный номер продукта иначе система не сможет найти его в базе данных и выдаст сообщение об ошибке. Также стоит учесть, что связать продукт с самим собой нельзя.


После внесения данных в поле нужно нажать на кнопку «Сохранить».
