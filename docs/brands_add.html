Добавление бренда

Чтобы добавить новый бренд нужно при помощи всплывающего меню OneBox, которое находится в левой части экрана, выбрать «Продукты» → «Бренды», затем вверху переключиться на вкладку «Добавить бренд», как на рисунке ниже:




После этого откроется страница «Создание новой записи», в которой есть все поля для создания нового бренда. Выглядит она так:




Из основных полей для создания брендов можно отметить следующие:




Имя бренда - единственное обязательное поле для создания бренда, без него система не сможет создать новую запись и выдаст ошибку.
Изображение — возможность загрузить логотип бренда.
Скрытый бренд — по умолчанию бренд виден в меню, если нужно его скрыть, то ставим галочку в checkbox`е.
Топ-бренд — наиболее популярный бренд, который будет отображаться на главной странице, если отметить checkbox галочкой.
Страна бренда — родина бренда.




Сайт бренда — ссылка на сайт о бренде.
Описание — служит для короткого пояснения к бренду.
После того как основные поля были заполнены, создать новый бренд можно, нажав на кнопку «Вставить новую запись», см. рисунок ниже:




Далее система напишет, что запись успешно добавлена и автоматически выдаст страницу для редактирования записи, чтобы можно было еще раз перепроверить все заполненные поля.
Если все данные были внесены корректно и не требуют каких-либо изменений, можно вернуться в список брендов, нажав на соответствующую вкладку вверху экрана.
