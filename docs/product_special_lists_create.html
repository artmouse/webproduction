Списки продуктов

Для объединения продуктов в списки нужно изначально создать списки. За это в системе отвечает отдельный функционал, который находится в «Продукты» → «Списки продуктов». Активируйте выдвижную панель в левой части экрана, затем нажмите левой кнопкой мышки на пункт «Продукты».




Далее найдите пункт «Списки продуктов».




После этого на экране появится страница всех списков, в которые были сгруппированы различные продукты. Выглядит она так:




Также на данную страницу можно попасть со страницы всех продуктов. Для этого нужно воспользоваться вкладкой «Списки продуктов», которая находится в верхней части экрана:




Колонка «Продукты в списках» показывает, какие списки существуют на данный момент.
Для того, чтобы начать работу со списками нужно нажать на одно из наименований в колонке.
После того как список был выбран, произойдет перезагрузка страницы и слева от колонки появятся все товары, которые входят в данный список.
Таблица товаров позволяет просмотреть все поля товаров. Если информация не помещается на экране, воспользуйтесь полосой прокрутки.


При помощи заголовков можно сортировать товары. Сортировка происходит по нажатию на заголовок колонки в верхней части страницы. Чтобы выполнить сортировку в обратном порядке нужно повторно нажать на заголовок.
Внизу таблицы находится пагинатор, который позволяет листать товары постранично. Для этого следует выбрать номер страницы или воспользоваться кнопками «Назад» и «Вперед» для перехода на предыдущую/следующую страницу соответственно.




При наведении курсора на заголовок списка изображение курсора изменится на знак «?». Это означает, что в этот момент пользователь может перейти к настройкам отображения таблицы - для этого нажмите правую кнопку мышки.


Страница настроек состоит из нескольких блоков и выглядит следующим образом:




В правом верхнем углу находится знак «Х», который служит для возврата на предыдущую страницу без сохранения изменений.
Первый блок позволяется регулировать отображение колонок. Он представлен множеством checkbox`ов, каждый их которых отвечает за свою колонку в таблице. Если checkbox отмечен, значит колонка отображается, если нет — колонка в таблице отображаться не будет. Важно! Это работает не для всех колонок, некоторые ключевые колонки должны быть видимыми всегда: их checkbox`ы не реагируют на клики.


Следующий блок представлен полями:
«Показывать записей на странице» - отображение количество записей на странице. Максимально допустимое значение - 100 записей.
«Сортировка по умолчанию» - выбор колонки, по которой будет происходить сортировка, также возможность сортировать по порядку или в обратном алфавитном порядке.


Внизу расположены кнопки, которые позволяют завершить работу с формой:
«Сохранить» - нажмите, чтобы изменения вступили в силу.
«Отмена» - возврат на предыдущую страницу без сохранения изменений.


Названия товаров являются ссылками. По нажатию на ссылку происходит переход к карточке товара.
Чтобы добавить в список новый товар нужно активировать панель в левой части экрана нажатием мышки. После этого панель будет выглядеть так:




В поле «Добавить еще продукты к списку (укажите коды (id) продуктов)» нужно ввести номер или id продукта, который должен быть добавлен в список. Для подтверждения изменений нажмите на кнопку «Сохранить».


Для удаления продукта из списка нужно отметить его галочкой в поле checkbox. После этого в панели быстрого редактирования должен появиться пункт «Удалить отмеченные продукты из списка» - отметьте его, затем нажмите на кнопку «Сохранить».




Для управления списками служит вкладка «Управление списками». Находится она в верхней части экрана, как изображено на рисунке:




Страница со всеми списками выглядит так:




Таблица позволяет выполнять сортировки при помощи панели фильтрации, которая активна по умолчанию. Для этого служат два поля: «Код» и «Наименование».




Введите в поле название списка, а для того, чтобы система выполнила фильтрацию воспользуйтесь кнопкой «Фильтровать», которая появится справа от панели.




Названия списков являются ссылками. По нажатию на ссылку происходит переход к форме редактирования списка. Данная форма обладает аналогичным набором полей, что и форма для добавления списка.
Чтобы добавить новый список нужно кликнуть на вкладку «Добавить список».




Форма для добавления/редактирования списка выглядит так:




«Название списка» - является обязательным для заполнения полем.
«Показывать на главной» - отвечает за отображение списка на главной странице сайта.
«Скрытый» - сделать список невидимым.
«Способ отображения» - разные способы отображения списка, важно, если список является видимым и отображается на главной странице.
«Автоматическая прокрутка» - опция задается только для карусели.


После заполнения всех полей для вставки новой записи нажмите на кнопку «Вставить новую запись» внизу экрана.




При использовании формы редактирования записи для сохранения изменений кликните на кнопку «Сохранить запись», если требуется удалить список — воспользуйтесь кнопкой «Удалить запись».
